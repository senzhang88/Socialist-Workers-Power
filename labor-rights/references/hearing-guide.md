# 仲裁开庭指南

## 庭前准备清单

### 一周前
- [ ] 熟读自己的仲裁申请书，确保每个请求都能说清来龙去脉
- [ ] 整理证据原件，按证据清单顺序排列
- [ ] 打印证据材料 2-3 套（仲裁庭一套、对方一套、自留一套）
- [ ] 准备一份开庭发言提纲（3-5 分钟能说完）
- [ ] 想好公司可能的抗辩理由和你的反驳

### 前一天
- [ ] 确认开庭时间、地点（一般上午 9 点或下午 2 点）
- [ ] 带齐：身份证原件、证据原件、仲裁申请书副本、纸笔
- [ ] 着装整洁（不需要西装，但要正式）
- [ ] 休息好，保持冷静

### 当天
- [ ] 提前 15-30 分钟到达
- [ ] 找到仲裁庭，签到
- [ ] 关手机或静音

## 开庭全流程详解

### 环节一：核对身份（5分钟）
仲裁员核对双方身份信息。
- 出示身份证原件
- 确认对方出庭人员身份（是否是公司的法定代表人或委托代理人）

### 环节二：宣布权利义务（5分钟）
仲裁员宣读仲裁庭纪律和双方权利义务。
- 有权申请回避（如果你认为仲裁员和对方有关系）
- 有权举证、质证、辩论
- 听着就行，不需要回应

### 环节三：申请人陈述（5-10分钟）
你的主场。按这个结构说：
1. **一句话概括**：我在什么时间入职公司，做什么工作，公司对我做了什么
2. **按时间线讲事实**：不要跳来跳去
3. **说明违法之处**：公司的行为违反了哪条规定
4. **重申请求**：所以我要求公司支付 XXX 元

**发言技巧：**
- 声音洪亮，语速适中
- 说事实，不要评价（"公司太无耻了" → 改为"公司未提前30天通知就口头辞退了我"）
- 每个关键事实都说"详见证据X"
- 不要和对方眼神接触，只看仲裁员

### 环节四：被申请人答辩（5-10分钟）
公司这边的说法。认真听，记笔记。
- 他们否认了什么？
- 他们提出了什么理由？
- 他们有什么证据？
- 他们的说法里有什么漏洞？

### 环节五：举证质证（最关键的环节，20-40分钟）

**你举证，对方质证：**
- 仲裁员："申请人，请出示你的证据"
- 你：拿出证据，逐一说明"证据X，证明XX事实"
- 对方会对你的证据提出质疑
- 你可以回应对方的质疑

**对方举证，你质证：**
- 对方出示证据
- 你来质证，从三个角度质疑：
  1. **真实性**："这是复印件，我方要求查看原件" / "这份文件我没有见过，不认可真实性"
  2. **合法性**："这份证据的取得方式不合法" / "这是公司单方面制作的我没有签字确认"
  3. **关联性**："这份证据和我主张的事实没有关系"

**质证金句：**
- "对这份证据的真实性不予认可"
- "这份证据系被申请人单方制作，未经我方确认"
- "该证据恰好证明了我方的观点"
- "对方提供的证据恰恰说明..."
- "这恰恰印证了公司确实存在XX行为"

### 环节六：辩论（10-20分钟）
双方就争议焦点展开辩论。仲裁员会归纳争议焦点，双方围绕焦点辩论。

**辩论策略：**
- 抓住对方漏洞不放
- 用事实和证据说话
- 不要被对方的情绪带跑
- 如果对方提出新说法，要求对方举证

### 环节七：最后陈述（各2-3分钟）
简短总结：
- "请仲裁庭支持我方的全部仲裁请求"
- 可以补充一句："事实很清楚，证据很充分，请求公正裁决"

### 环节八：再次调解
仲裁员会问双方是否愿意调解。
- 你可以同意尝试，但要有底线
- 也可以直接拒绝："我方不接受调解，请依法裁决"
- 如果调解达成一致，会当庭制作调解书
- 调解不成，等待裁决

## 常见公司抗辩及应对

| 公司抗辩 | 你的应对 |
|---------|---------|
| "他是主动辞职的" | 要求公司提供辞职信原件；出示录音/聊天记录证明是被辞退的 |
| "他严重违反规章制度" | 要求公司出示规章制度且证明你签字确认过；要求公司举证"严重"的标准 |
| "公司经营困难" | 经营困难不是违法解除的理由；合法裁员需要走法定程序 |
| "绩效不达标" | 要求出示考核标准和你签字确认的考核结果；不能胜任工作需要先培训或调岗 |
| "我们签了协商解除协议" | 检查协议是否是胁迫/欺骗签的；检查协议内容是否显失公平 |
| "考勤记录没有了" | 考勤记录属于用人单位保管，举证责任在公司 |
| "他不是我们的员工" | 出示所有劳动关系证据：工资流水、社保记录、工牌、工作群记录等 |

## 开庭心态管理

- **紧张是正常的**——第一次上庭谁都紧张
- **仲裁员不是法官**——劳动仲裁相对没那么严肃，更像调解
- **说错了可以更正**——不要因为紧张说错了就慌
- **不要和对方吵架**——只和仲裁员沟通，对方说什么你质证就好
- **没说完的可以补充书面意见**——开庭后一般还有时间提交补充材料

## 庭后

- 仲裁员会告知裁决时间（一般 45 天内）
- 如果想到新的证据或观点，可以写"代理词"提交给仲裁委
- 等待期间不要和公司私下协商（除非你愿意）
- 收到裁决书后，15天内决定是否起诉
