# 入职避雷指南 - 劳动者自我保护手册

> 本指南帮助劳动者在入职前识别潜在风险，避免掉入"黑心企业"陷阱。

---

## 一、企业背景调查（入职前必查）

### 1.1 天眼查/企查查使用指南

**必查项目清单：**

| 项目 | 如何查 | 风险信号 |
|------|--------|---------|
| **经营状态** | 看"企业状态" | 显示"注销""吊销""清算"不要入职 |
| **注册资本** | 看"注册资本" vs "实缴资本" | 认缴很高但实缴为0，可能是空壳 |
| **成立时间** | 看"成立日期" | 成立不到1年的初创公司风险高 |
| **涉诉信息** | 看"司法风险" | 大量劳动纠纷、被执行案件 |
| **经营异常** | 看"经营异常" | 地址异常、年报异常 |
| **股权冻结** | 看"股权出质/冻结" | 股东股权被冻结，资金链可能断裂 |
| **变更记录** | 看"变更记录" | 频繁变更法人、股东、地址 |

**实操步骤：**

1. **搜索企业全称**（注意不是简称）
2. **查看风险等级**（天眼查会显示"低风险/中风险/高风险"）
3. **重点看劳动纠纷**：在"司法风险"里搜索"劳动争议"
4. **看员工评价**：企查查有"员工点评"（虽主观但可参考）

---

### 1.2 面试时的观察清单

**办公环境观察：**

| 观察点 | 正常情况 | 异常情况 |
|--------|---------|---------|
| **办公地点** | 正规写字楼 | 居民楼、虚拟地址 |
| **办公面积** | 与员工数量匹配 | 员工多但办公面积很小 |
| **员工状态** | 正常办公 | 死气沉沉、大量空位 |
| **公司名牌** | 有正规标识 | 没有或很简陋 |
| **面试流程** | 规范、专业 | 随意、催促入职 |

**面试官话术识别：**

| 话术 | 潜台词 | 风险等级 |
|------|--------|---------|
| "我们是创业公司，看重潜力" | 薪资可能低于市场价 | ⚠️ 中 |
| "试用期工资打8折，转正后补" | 转正可能不补 | ⚠️⚠️ 高 |
| "先干着，合同后面再签" | 可能不签合同 | ⚠️⚠️⚠️ 极高 |
| "我们实行大小周/单休" | 加班费可能不给 | ⚠️⚠️ 高 |
| "工资发现金，避税" | 社保可能按最低基数缴 | ⚠️⚠️ 高 |
| "竞业限制是标准条款，大家都签" | 离职后可能受限 | ⚠️⚠️ 高 |

## 二、劳动合同审查要点

### 2.1 合同必查条款

**签合同前必须逐条确认：**

#### 1. 薪资条款（最关键）

| 要确认的点 | 风险描述 | 正确写法 |
|-----------|---------|---------|
| **薪资构成** | 只说"月薪1万"但不写构成 | 基本工资+绩效+补贴分别写明 |
| **试用期工资** | 低于转正工资80% | 不得低于转正工资的80% |
| **发薪日** | 写"每月下旬"不明确 | 明确到"每月X日" |
| **年终奖** | 写"视公司经营情况" | 明确计算方式或至少写"按制度执行" |

**计算公式：**
```
试用期最低工资 = 转正工资 × 80%
例如：转正10,000元 → 试用期不得低于8,000元
```

#### 2. 试用期条款

| 合同期限 | 法定试用期上限 | 超过即违法 |
|---------|---------------|-----------|
| 3个月-1年 | 1个月 | ⚠️ |
| 1年-3年 | 2个月 | ⚠️ |
| 3年以上 | 6个月 | ⚠️ |
| 无固定期限 | 6个月 | ⚠️ |

**注意：**
- 同一单位与同一劳动者**只能约定一次试用期**
- 试用期包含在劳动合同期限内
- 试用期不得低于当地最低工资标准

#### 3. 工作地点与岗位

| 风险条款 | 正确做法 |
|---------|---------|
| "工作地点：全国"或"公司指定地点" | 要求明确到城市，或约定变更需协商 |
| "岗位：公司根据经营需要调整" | 要求明确岗位名称，调岗需协商一致 |
| "公司有权单方面调整工作内容" | 违法，要求删除 |

#### 4. 竞业限制条款（高风险）

**如果合同里有竞业限制，必须确认：**

| 要素 | 必须明确 | 风险 |
|------|---------|------|
| **限制范围** | 明确列出竞争公司名单或行业 | 范围过宽无法工作 |
| **限制地域** | 具体到城市或区域 | "全国"范围无效 |
| **限制期限** | 最长2年 | 超过2年违法 |
| **补偿金标准** | 写明离职后每月支付金额 | 未约定则公司可能不付 |
| **违约金** | 明确违约赔偿标准 | 过高可能被法院调低 |

**上海标准：**
- 竞业限制补偿金：一般不低于离职前12个月平均工资的30%
- 如果低于最低工资，按最低工资支付

#### 5. 其他高风险条款

**看到以下条款要警惕：**

❌ **"自愿放弃社保"** - 违法无效
❌ **"工伤自负"** - 违法无效
❌ **"加班不给加班费"** - 违法
❌ **"迟到一次扣半天工资"** - 处罚过重
❌ **"入职押金/服装费"** - 违法
❌ **"离职需提前3个月申请"** - 试用期3天/转正30天才是法定

---

### 2.2 合同签署注意事项

**签署前：**
- [ ] 要求合同一式两份，自己保留一份原件
- [ ] 合同所有空白处必须填好或划掉
- [ ] 手写修改处必须双方签字确认
- [ ] 页脚必须有页码，防止换页

**签署时：**
- [ ] 逐页签字或盖骑缝章
- [ ] 签署日期必须是实际入职日期
- [ ] 确保公司盖章清晰

**签署后：**
- [ ] 立即拍照/扫描备份
- [ ] 存放到云盘+本地双备份
- [ ] 合同和offer/邮件等其他证据一起保存

## 三、入职第一周的自我保护

### 3.1 必须保留的证据

**入职当天就要开始做：**

| 证据类型 | 如何保存 | 用途 |
|---------|---------|------|
| **劳动合同** | 原件+扫描件 | 证明劳动关系、工资标准 |
| **录用通知书（offer）** | 邮件截图 | 证明入职条件、薪资承诺 |
| **员工手册** | 签收页拍照 | 审查制度合法性 |
| **工牌/门禁卡** | 拍照 | 证明劳动关系 |
| **第一天考勤记录** | 截图 | 证明入职时间 |
| **工作群聊天记录** | 截图 | 证明实际工作内容 |

### 3.2 立即确认的事项

**入职一周内必须搞清楚：**

- [ ] 工资发放日（具体几号）
- [ ] 社保公积金缴纳基数（是否按实际工资）
- [ ] 试用期考核标准（量化指标）
- [ ] 年假天数及计算方式
- [ ] 加班审批流程
- [ ] 离职流程（需要提前多久申请）

**确认方式：**
- 邮件或书面确认（不要只问HR口头）
- 保存HR回复的截图/邮件

## 四、常见陷阱识别

### 4.1 "合法但不合理"的套路

| 套路 | 是否违法 | 如何应对 |
|------|---------|---------|
| **试用期不给缴社保** | ❌ 违法 | 入职当月就要缴，可投诉 |
| **按最低基数缴社保** | ❌ 违法 | 要求按实际工资缴纳 |
| **工资拆分为基本工资+绩效** | ⚠️ 灰色 | 要求明确绩效计算方式 |
| **签劳务合同而非劳动合同** | ❌ 可能是违法用工 | 实质是劳动关系的，可主张 |
| **要求签"自愿加班声明"** | ❌ 违法 | 拒绝签署，保留证据 |
| **年终奖写"视效益发放"** | ⚠️ 模糊条款 | 要求明确计算方式 |

### 4.2 互联网/科技公司特有风险

| 风险点 | 识别方法 | 防范措施 |
|--------|---------|---------|
| **PIP绩效改进陷阱** | 合同写"不能胜任可解除" | 要求明确考核标准 |
| **996加班文化** | 面试时说"弹性工作" | 明确加班费计算 |
| **期权画饼** | 口头承诺期权 | 要求写入合同 |
| **竞业限制滥用** | 全员签竞业限制 | 确认补偿金标准 |
| **末位淘汰** | 制度写"末位淘汰" | 违法，可主张赔偿 |

## 五、紧急情况应对

### 5.1 入职后发现被骗

**情况1：实际薪资与offer不符**
- 立即书面提出异议
- 要求按offer执行或解除合同并赔偿
- 保留offer、合同、沟通记录

**情况2：公司不签合同**
- 超过1个月未签合同：可主张双倍工资
- 超过1年未签合同：视为无固定期限合同
- 注意收集考勤、工资流水等证据

**情况3：公司实际经营异常**
- 立即更新简历准备跳槽
- 同时收集证据以备仲裁
- 如果拖欠工资，立即投诉

### 5.2 快速判断公司是否靠谱

**3分钟快速评估法：**

| 评估项 | 得分 | 标准 |
|--------|------|------|
| 天眼查风险等级 | -2~+2 | 高风险-2，中风险-1，低风险+2 |
| 社保缴纳情况 | -2~+2 | 入职当月缴+2，试用期后缴-2 |
| 合同规范性 | -2~+2 | 条款清晰+2，条款模糊-1 |
| 面试专业度 | -1~+1 | 专业+1，随意-1 |
| **总分** | **-7~+7** | **<0不建议入职，>3较靠谱** |

### 6.2 入职审查清单（打印版）

```
入职前审查清单

企业背景调查：
□ 天眼查/企查查风险等级
□ 涉诉信息（重点看劳动纠纷）
□ 经营状态（非注销/吊销）
□ 成立时间（非近期成立）

合同审查：
□ 薪资构成明确
□ 试用期符合法定上限
□ 工作地点明确
□ 岗位名称明确
□ 无"自愿放弃社保"条款
□ 竞业限制条款合理（如有）

入职证据保存：
□ 合同原件（自留一份）
□ 录用通知书（offer）
□ 员工手册签收页
□ 第一天考勤记录

确认事项：
□ 工资发放日（具体几号）
□ 社保缴纳基数
□ 试用期考核标准
□ 年假天数
```
