## 🔍 如何使用案例库

### 场景1：准备仲裁/诉讼

1. **确定案由**：到`by-case-type/`查找同类案件
2. **参考策略**：查看`analysis/win-patterns.md`了解胜诉规律
3. **避开陷阱**：查看`analysis/lose-pitfalls.md`了解常见败诉原因
4. **计算金额**：查看`analysis/compensation-calc.md`参考赔偿计算方式

### 场景2：研究特定公司

1. **查询公司**：到`by-company/`查看目标公司历史案例
2. **了解策略**：分析该公司常用抗辩理由和诉讼策略
3. **制定对策**：针对性准备证据和主张

### 场景3：研究特定地区

1. **查询地区**：到`by-region/`查看该地区裁判倾向
2. **了解法官**：分析该地区法院/仲裁委的裁判尺度
3. **调整策略**：根据地区特点调整维权策略

- 腾讯云OCR API

**文本提取**：
```python
# Python代码示例
import pdfplumber

def extract_text_from_pdf(pdf_path):
    with pdfplumber.open(pdf_path) as pdf:
        text = ""
        for page in pdf.pages:
            text += page.extract_text()
    return text
```

## ⚠️ 隐私与合规

### 脱敏要求

1. **当事人信息**：隐去真实姓名（用"李某""某公司"代替）
2. **公司信息**：隐去公司全称（用"某科技公司""互联网公司A"代替）
3. **敏感数据**：隐去身份证号、银行账号等
4. **联系方式**：隐去电话、地址等
