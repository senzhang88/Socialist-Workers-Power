# 民事诉讼证据规定-使用说明

## 为什么这里分成两份

最高法官网当前我核到的稳定官方来源是：

- [09-民事诉讼证据规定-2002原规定](./09-民事诉讼证据规定-2002.md)
- [09A-民事诉讼证据规定-2019修改](./09A-民事诉讼证据规定-2019修改.md)

也就是说，这一组文件在最高法官网上更稳定的保存方式，是“原规定 + 修改决定”，而不是我这次直接抓到了一份独立的“现行整合版”网页。

## 实际使用时怎么读

1. 先看 `09A`，确认 2019 年修改了哪些条文。
2. 再回到 `09`，核对原条文位置。
3. 如果你准备把某一条直接写进仲裁申请书、起诉状或代理词，建议最后再以最高法官网对应原页面复核一次条号和措辞。

## 当前案件里最有用的方向

- 证据提交期限
- 证据交换
- 非法证据/不当证据的质证思路
- 电子数据、录音录像、书证、鉴定意见的证明力

## 说明

这里不是内容不可靠，而是我故意把“原规定”和“修改决定”都存下来，避免把未经核对的第三方整合文本误当成官方现行文本。
