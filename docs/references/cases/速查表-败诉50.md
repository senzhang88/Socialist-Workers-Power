# 败诉50案例速查表

> **来源：上海市浦东新区人民法院劳动争议真实判决**
> 筛选标准：法院驳回劳动者诉求或赔偿金额极低（<1万元）
> 价值：警示劳动者避免常见错误

---

## ⚠️ 维权前自查清单

对照以下清单自查，避免落入真实案例反复出现的败诉陷阱：

- [ ] 我有**书面**证据证明加班/软裁员/违法解除吗？（无证据 = A类败诉）
- [ ] 我的仲裁时效还没过吗？离职是否已超过1年？（超时效 = C类败诉）
- [ ] 我的辞职信写的是"个人原因"吗？（写了 = D类败诉高风险）
- [ ] 我是否已经签收了任何离职协议？（签了 = D类败诉）
- [ ] 我有没有违纪行为被公司留存了记录？（有 = B类败诉风险）
- [ ] 我的证据是完整的截图还是部分截取？（不完整 = E类败诉）

### 最值得警惕的3个真实案例

**案例B1·张某某案**：员工被举报饭卡外借，公司有OA系统记录+餐费凭证，员工在谈话录音中亲口说"**我承认的！但到了仲裁的时候我不需要承认，我就说不承认**"——这句录音被公司提交，成为最致命证据。

**案例D1·宋某某案**：员工辞职信写"个人原因"，事后主张被迫离职索赔82万，被全额驳回。**辞职信一旦写"个人原因"，几乎无法翻案**。

**案例B2·郭某某案**：员工离职后窃取公司商业秘密，不仅仲裁败诉，还需赔偿公司50万。保密协议在离职后仍然有效。

## 维权避坑四步走

**入职时**：保留劳动合同原件 + offer原件 + 了解员工手册和年终奖制度

**在职时**：工资条/银行流水 + 加班须获审批 + 病假按流程申请并保留病假单

**离职时**：不写"个人原因" + 若被迫离职须写明具体理由 + 离职协议要核查年终奖条款

**维权时**：离职后1年内申请仲裁 + 证据须完整且符合形式 + 及时行动不拖延
