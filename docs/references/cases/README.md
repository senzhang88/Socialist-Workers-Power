## 🎯 设计逻辑

本案例库**不堆砌原始文书**，而是用100个精华案例分层呈现：

- **新用户** → 先看[高价值案例精选](高价值案例精选.md)（10分钟了解全局）
- **有具体场景** → 看按类型分类文件（针对性强，见下方导航）
- **想全面了解** → 看[速查表](速查表-胜诉50.md)（完整表格 + 关键词检索）
- **想避坑** → 看[败诉50](速查表-败诉50.md)（含维权前自查清单）
