# 医疗期保护 · 真实判例与法律规则

> **来源：上海市浦东新区人民法院真实判决**（586份裁判文书数据库）

## 医疗期基本规则

### 医疗期长度（按在本单位工龄计算）

| 工龄 | 医疗期 |
|------|--------|
| 不满2年 | 3个月 |
| 2年以上不满4年 | 6个月 |
| 4年以上不满6年 | 9个月 |
| 6年以上不满8年 | 12个月 |
| 8年以上不满10年 | 18个月 |
| **10年以上** | **24个月** |

> 上海规定：与本单位工龄满20年且年龄满50岁（女）或55岁（男），医疗期为24个月。

### 病假工资标准（上海）

| 连续工龄 | 病假工资比例 |
|----------|-------------|
| 不满2年 | 合同工资的60% |
| 2年以上不满4年 | 合同工资的70% |
| 4年以上不满6年 | 合同工资的80% |
| 6年以上不满8年 | 合同工资的90% |
| **8年以上** | **合同工资的100%** |

> 最低不得低于上海市最低工资标准的80%。

## 📋 真实案例

### 案例1：叶某 vs 某某公司

**案号**：（2024）沪0115民初105902号
**案件类型**：调岗争议 + 病假工资差额（医疗期保护）

**核心事实**：
- 叶某在公司工龄27年，对应医疗期24个月，连续工龄8年以上
- 公司项目落标后，单方要求叶某从事新项目工作（实质调岗）
- 叶某对调岗持异议，其后申请病假
- 公司发放病假工资远低于法定标准

**法院认定**：
- 连续工龄8年及以上，**病假工资按合同工资100%计发**（6个月内）
- 超过6个月，降至60%计发
- 公司欠付病假工资差额 **43,592.40元**

**裁决结果**：支持补发，案件合计赔偿 **100,381.83元**（含加班费及绩效差额）

**关键规则**：
- 病假工资须严格按法定比例支付，"公司认为多少合适"不是法律标准
- 工龄越长比例越高，8年以上工龄100%全额发放（6个月内）

### 案例2：李某某 vs 中国东方航空股份有限公司

**案号**：（2022）沪0115民初19122号
**案件类型**：待岗 + 医疗期 + 违法解除
**赔偿金额**：**570,603.21元** ⭐高额赔偿

**核心事实**：员工处于医疗期内，公司以"组织优化"为由安排待岗并停发工资，同时涉及年终奖争议。

**法院认定**：
- 医疗期内不得违法解除，公司须承担违法解除赔偿金
- 待岗期间工资须**全额发放**（不能仅发最低工资）
- 年终奖须按实际履职期间支付

**关键规则**：医疗期 + 软裁员（待岗降薪）叠加时，公司须承担**双重赔偿责任**，是劳动争议中赔偿金额最高的组合之一。

### 案例3：潘家敏 vs 上海怡康化工材料有限公司

**案号**：（2018）沪0115民初
**案件类型**：病假期间解除 → 违法

**案情**：员工病假期间，公司以"病假不成立"为由解除劳动关系。员工向法院申请恢复劳动关系。

**法院认定**：公司解除违法，支持恢复劳动关系或支付违法解除赔偿金（2N）。

**关键规则**：
- 医疗期内，公司**不得以"不能胜任工作"为由**解除
- 对病假是否成立有异议，须通过合法途径（要求补充医院证明等）处理，不能直接解除

### 案例4：吴某 vs 某某公司（程序失误案例）

**案号**：（2025）沪0115民初63164号
**案件类型**：医疗期内违法解除 → **裁定不予受理（程序错误）**

**核心事实**：
- 吴某确诊脊髓肿瘤，医疗期刚开始（2023年11月），公司即以"客观情况重大变化"为由通过邮件发出解除通知
- 公司曾书面承诺支付补偿金等317,432.11元，但一直未履行
- 吴某在仲裁时仅列某某公司1为被申请人，起诉时追加了母公司某某公司2
- 法院裁定：**仲裁时未列某某公司2，不符合仲裁前置程序，不予受理**

!!! danger "仲裁前置程序铁律"
    仲裁申请书中**必须列明所有相关被申请人**（包括关联公司、用工单位、劳务派遣方）。

    遗漏后在诉讼阶段追加，法院以"该公司未经仲裁程序"为由裁定不予受理。

    **起诉前务必查清**：用工单位、劳动合同签约方、劳务派遣方，**全部列入仲裁申请**。

### 案例5：马某某 vs 某某公司（败诉案例）

**案号**：（2018）沪0115民初94337号
**案件类型**：主张医疗期保护 → **全额败诉**

**核心事实**：
- 员工主张医疗期保护，但其医疗期已于2018年6月30日届满
- 公司在2018年5月通知不续签合同，通知时员工**已不在医疗期内**

**法院认定**：医疗期届满后不再享有医疗期特殊保护，诉求驳回。

**教训**：须主动核算自己的法定医疗期长度（按**在本单位**的工龄计算，非总工龄），避免误判保护范围。

## 💡 医疗期维权要点

### 生病时立即做的事

!!! tip "医疗期保护四步走"
    1. **及时请假**：第一时间通过书面方式（微信/邮件）通知用人单位，留存截图
    2. **正规医院就诊**：二甲及以上医院开具病假证明
    3. **核对工龄**：确认自己在**本单位**的连续工龄，对应法定医疗期长度
    4. **核查工资**：每月核对病假工资是否按法定比例支付

### 病假工资自测公式

```
连续工龄不满2年：合同工资 × 60%
连续工龄2年以上不满4年：合同工资 × 70%
连续工龄4年以上不满6年：合同工资 × 80%
连续工龄6年以上不满8年：合同工资 × 90%
连续工龄8年以上：合同工资 × 100%（前6个月）
         超过6个月后：合同工资 × 60%

最低不得低于上海最低工资标准的80%
```

### 公司催促复岗时的应对

- 核实自己的医疗期是否已届满
- 医疗期未满：书面回复"医疗期未届满，依法享有休息权利"
- 医疗期届满但身体仍不适：申请劳动能力鉴定
- 收到解除通知：立即咨询，判断是否违法

### 医疗期满合法解除的底线

即使公司合法解除，你仍应获得：
- **N**（经济补偿，按工龄计算）
- **医疗补助金**（一般情形不低于6个月工资；重病不低于12个月；绝症不低于18个月）
- **+1代通知金**（如果没有提前30天书面通知）
