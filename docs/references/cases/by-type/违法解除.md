# 违法解除 · 10个真实案例

> **来源：上海市浦东新区人民法院真实判决**
> 举证不能 / 程序违法类违法解除

---

## 合法解除 vs 违法解除

| 解除类型 | 法律依据 | 赔偿 |
|----------|----------|------|
| 协商一致解除 | 第36条 | N（经济补偿） |
| 员工过失（有证据）| 第39条 | 0 |
| 非过失性解除（病/不胜任等）| 第40条 | N+1 |
| 经济性裁员 | 第41条 | N |
| **违法解除** | **第48条** | **2N（赔偿金）** |

**关键**：违法解除赔偿金（2N）= 月均工资 × 工龄年数 × 2

### 案例9：潘某某案 · 赔偿155,808元

**案号**：（2017）沪0115民初72914号
**原始文件**：681_宾德环境试验设备(上海)有限公司与潘某某劳动合同纠纷一审民事判决书.md

**案情**：公司主张员工利益冲突、在外设公司，单方解除劳动关系。员工要求恢复劳动关系+年终奖+13薪。

**法院认定**：
> "公司举证不能，解除违法。原告的离职并非其本人原因所致，应按约定支付年终奖和13薪。"

**裁决**：违法解除赔偿金106,290元 + 年终奖32,847元 + 13薪16,671元，合计 **155,808元**

**典型意义**：公司主张员工有利益冲突或外设公司，**必须举证**，举证不能则解除违法。同时，违法解除后年终奖的在职条件视为已成就。

## 💡 从10个真实案例归纳的规律

### 规律1：违法解除案件中，公司举证不能是最大原因

上述案例（案例1-8）全部因**公司举证不能**而败诉。公司主张解除须承担举证责任，需提供：

- 明确的规章制度条文（需有员工知晓证明）
- 违纪事实的书面记录
- 处分通知书（有员工签字确认）

**缺一不可**。

### 规律2：赔偿金计算公式

```
违法解除赔偿金 = 月均工资 × 工龄年数 × 2
月均工资：离职前12个月平均工资
工龄：满1年算1年；6个月以上不满1年算1年；不满6个月算0.5年
注意：若月均工资超过本市上年度职工月平均工资3倍，以3倍封顶
```

### 规律3：违法解除后年终奖仍须支付

潘某某案确立原则：公司违法解除，年终奖"发放时在职"的条件视为已成就——是公司的违法行为导致员工无法在发放日在职，责任在公司。

### 收到解除通知后的第一步

!!! tip "立即做这三件事"
    1. **不要签任何文件**（尤其是"协商一致解除协议"或"离职协议"）
    2. **保存解除通知原件或截图**
    3. **记录时间节点**（口头通知、书面通知、实际离岗日期）
