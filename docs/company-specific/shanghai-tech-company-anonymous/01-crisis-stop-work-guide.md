# 上海某互联网公司停岗待业应对指南

> **紧急版本 v1.0** | 适用场景：被口头通知停岗/待岗/不让进公司
> 
> **核心理念**：停岗待业是违法解除劳动合同的"软手段"，目标是逼你自己辞职。坚持住，2N赔偿金是你的法定权利。

---

## 📍 第一步：立即确认公司行为性质（今天完成）

### 识别信号清单

| 公司行为 | 法律性质 | 你的权利 |
|---------|---------|---------|
| 口头通知"停岗待业" | ❌ 违法（需书面通知） | 要求书面通知，录音取证 |
| 不让进入办公区/门禁失效 | ❌ 违法解除劳动关系 | 报警留证，主张违法解除 |
| 飞书/钉钉被禁言/踢出群 | ❌ 违法解除劳动关系 | 截图录屏，保存证据 |
| 收回电脑/工牌 | ❌ 违法解除劳动关系 | 拍照/录像记录 |
| 要求签署"自愿离职" | ❌ 违法（胁迫签署无效） | 拒绝签署，要求2N赔偿 |
| 停发工资但社保继续缴纳 | ⚠️ 待岗状态（需协商一致） | 要求恢复劳动条件或2N赔偿 |

### 关键法律认知

**停岗待业的法律真相**：

1. **公司单方面停岗 = 违法**
   - 《劳动合同法》第38条：未按照劳动合同约定提供劳动条件的，劳动者可以解除劳动合同
   - 《劳动合同法》第48条：用人单位违法解除劳动合同，劳动者可以要求继续履行或2N赔偿金

2. **停岗期间工资标准**
   - 第一个月：全额工资
   - 第二个月起：最低工资（上海2025年：2,690元/月）
   - **但**：如果公司恶意停岗，你可以主张全额工资差额

3. **你的两条维权路径**
   - **路径A（推荐）**：主张违法解除，要求2N赔偿金
   - **路径B**：要求恢复劳动关系，补发停岗期间工资

#### 2. 录音取证关键话术

**如果对方口头通知停岗，立即录音并要求确认以下信息**：

```
"今天是___年___月___日，您是___（职位）___（姓名）吗？
```

#### 3. 发送《要求恢复劳动条件通知书》

**通过以下渠道发送**（全部都要）：
- ✅ 企业邮箱（发送给自己个人邮箱备份）
- ✅ 钉钉/飞书（截图保存送达记录）
- ✅ 微信工作群（截图保存送达记录）
- ✅ EMS邮政特快专递（保留底单，查询签收记录）

**通知书模板**：

```
要求恢复劳动条件通知书

致：___公司

本人___（姓名），身份证号___，于___年___月___日入职贵司，担任___职务。

___年___月___日，贵司口头通知本人"停岗待业"，并阻止本人进入办公区域/收回工牌/禁用飞书账号（选择适用项）。

根据《劳动合同法》第38条，贵司未按照劳动合同约定提供劳动条件，已构成违法。现正式通知贵司：

1. 要求贵司立即恢复本人的劳动条件，包括：提供办公场地、恢复系统权限、正常安排工作；
2. 如贵司无法恢复劳动条件，视为贵司违法解除劳动合同，本人将依法主张2N赔偿金；
3. 在劳动条件恢复前，本人仍按时出勤，如贵司阻止本人进入办公场所，一切法律后果由贵司承担。

特此通知。

申请人：___（签名）
日期：___年___月___日
```

### 【本周内完成】3天内

#### 1. 证据整理与备份

**核心证据清单**：

| 类别 | 具体内容 | 保存方式 |
|------|---------|---------|
| **劳动关系** | 劳动合同、offer、工牌、考勤记录 | 原件+扫描件+云盘备份 |
| **工资收入** | 近12个月工资流水、工资条、个税记录 | 银行流水+云盘备份 |
| **停岗证据** | 门禁失效录像、报警记录、聊天记录 | 本地+云端双备份 |
| **公司制度** | 员工手册、考勤制度、绩效制度 | 截图/拍照+备份 |
| **沟通记录** | 与HR/主管的所有微信/钉钉/邮件 | 截图+录屏+备份 |

#### 2. 关键信息记录

记录以下信息（用于后续仲裁）：

- [ ] 入职日期：____年____月____日
- [ ] 离职日期（停岗日期）：____年____月____日
- [ ] 工作年限：____年____月 = ____年（用于计算N）
- [ ] 近12个月平均工资：____元（税前）
- [ ] 月薪标准：____元（合同约定）
- [ ] 年终奖/绩效：____元/年
- [ ] 公司全称：____
- [ ] 注册地址：____
- [ ] 实际办公地址：____

#### 3. 赔偿金预估

**计算公式**：

```
工作年限N = 每满1年算1年，6个月以上算1年，不满6个月算0.5年

经济补偿金（N）= N × 离职前12个月平均工资

违法解除赔偿金（2N）= 2 × N × 离职前12个月平均工资

注意：若平均工资超过上海社平工资3倍（37,302元/月），按37,302元封顶计算
```

**示例**：

| 工作年限 | 月平均工资 | 2N赔偿金 |
|---------|-----------|---------|
| 3年 | 15,000元 | 90,000元 |
| 5年 | 20,000元 | 200,000元 |
| 5年（工资5万，超3倍封顶） | 37,302元 | 373,020元 |

## 📍 第三步：与公司沟通策略

### ❌ 绝对不要做的事

| 不要做 | 后果 |
|--------|------|
| 签署"自愿离职""个人原因离职" | 失去2N赔偿权利 |
| 签署"协商解除"但未谈好金额 | 可能只能拿N |
| 情绪激动，与公司冲突 | 公司可能以此为由合法解除 |
| 缺勤/旷工 | 公司可能以严重违纪合法解除 |
| 破坏公司财物/删除数据 | 公司可能合法解除并索赔 |
| 在社交媒体公开攻击公司 | 影响后续谈判/仲裁 |

### ✅ 应该做的事

| 要做 | 目的 |
|------|------|
| 每天按时到公司打卡/拍照 | 证明你要求履行劳动合同 |
| 所有沟通录音/截图 | 保留证据 |
| 书面回复公司所有通知 | 留下书面痕迹 |
| 保持冷静，理性沟通 | 展现专业，为仲裁加分 |
| 咨询律师或拨打12348 | 获取专业建议 |
| 准备仲裁材料 | 做好最坏的打算 |

### 💬 谈判话术参考

**当HR说"这是公司的决定，你必须接受"时**：

```
"我理解公司有经营考虑，但这个决定涉及我的合法权益。
根据劳动合同法，公司单方面停岗是违法的。
我可以接受协商解除，但赔偿标准需要依法计算。
我的诉求是2N赔偿金，共计___元。
如果公司无法接受，我只能通过法律途径维权。"
```

**当HR威胁"仲裁会影响你下一份工作"时**：

```
"我有权依法维权。仲裁记录不公开，不会影响我的职业发展。
如果公司愿意依法赔偿，我们可以友好协商解决。
但违法解除劳动合同的事实是明确的。"
```

## ⚠️ 重要提醒

1. **时效不等人**：劳动仲裁申请时效为1年，从离职之日起算，超过时效将无法维权

2. **证据是核心**：没有证据的仲裁申请很难获胜，务必在停岗第一时间固定证据

3. **坚持就是胜利**：公司通常会采取"拖"字诀，可能持续1-2年，但只要你坚持，最终大概率能拿到赔偿

4. **寻求专业帮助**：如案情复杂，建议咨询专业劳动法律师或拨打12348法律援助热线

5. **保护个人信息**：维权过程中注意保护个人隐私，不要随意在社交媒体暴露敏感信息
