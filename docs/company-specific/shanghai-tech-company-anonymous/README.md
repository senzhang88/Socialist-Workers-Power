## 💡 快速计算

### 2N赔偿金速算

```
工作年限N = 每满1年算1年，6个月以上算1年，不满6个月算0.5年

2N赔偿金 = 2 × N × 月平均工资

注意：月平均工资超过37,302元的，按37,302元封顶计算
```

### 示例

| 工作年限 | 月平均工资 | 2N赔偿金 |
|---------|------------|---------|
| 3年 | 15,000元 | 90,000元 |
| 5年 | 20,000元 | 200,000元 |
| 5年（工资5万，超封顶） | 37,302元 | 373,020元 |

## ⚠️ 重要提醒

1. **时效不等人**：劳动仲裁申请时效为1年，从离职之日起算
2. **证据是核心**：没有证据的仲裁申请很难获胜
3. **坚持就是胜利**：公司通常会采取"拖"字诀，但只要你坚持，最终大概率能拿到赔偿
4. **寻求专业帮助**：如案情复杂，建议咨询专业劳动法律师或拨打12348

## 🔑 核心法条

**《劳动合同法》第38条**：
> 用人单位未按照劳动合同约定提供劳动条件的，劳动者可以解除劳动合同。

**《劳动合同法》第48条**：
> 用人单位违法解除劳动合同，劳动者可以要求继续履行或2N赔偿金。

**《劳动合同法》第87条**：
> 用人单位违法解除或终止劳动合同的，应当依照本法第四十七条规定的经济补偿标准的**二倍**向劳动者支付赔偿金。
