# 证据收集清单

> **证据是仲裁成败的关键。本清单帮助你系统性收集和整理证据。**

---

## 📊 证据分级体系

### ⭐⭐⭐ 核心证据（必备，没有很难赢）

| 证据 | 用途 | 收集方式 | 保存要求 |
|------|------|---------|---------|
| **门禁失效证据** | 证明公司阻止你工作 | 刷卡失败录像，显示时间地点 | 原始视频+云端备份 |
| **报警记录** | 证明你要求履行合同 | 报警后索要出警回执 | 原件扫描+拍照 |
| **系统被禁证据** | 证明公司切断工作条件 | 飞书/钉钉被禁言/踢出录屏 | 原始载体+云端备份 |
| **书面停岗通知** | 证明公司违法解除 | 要求公司出具盖章书面通知 | 原件+扫描件 |
| **劳动合同** | 证明劳动关系 | 找出入职时签署的原件 | 原件+扫描件 |

### ⭐⭐ 重要证据（强烈建议收集）

| 证据 | 用途 | 收集方式 | 保存要求 |
|------|------|---------|---------|
| **工资流水** | 计算赔偿金基数 | 银行APP导出近12个月 | 原始PDF+截图 |
| **工资条** | 证明工资构成 | 收集历史工资条/邮件 | 原件+扫描件 |
| **考勤记录** | 证明实际工作 | 从考勤系统导出/截图 | 原始文件+备份 |
| **入职通知/offer** | 证明入职时间/岗位 | 找出邮件/纸质offer | 截图+扫描件 |
| **工牌/名片** | 证明劳动关系 | 拍照保存 | 高清照片+备份 |
| **社保缴纳记录** | 证明工作年限 | 上海人社APP导出 | 截图+PDF |
| **纳税记录** | 证明收入 | 个人所得税APP导出 | 截图+PDF |

### ⭐ 补充证据（有则更好）

| 证据 | 用途 | 收集方式 |
|------|------|---------|
| 工作邮件往来 | 证明工作内容 | 导出重要邮件 |
| 工作成果/项目文档 | 证明工作表现 | 整理项目资料 |
| 同事证言 | 证明停岗事实 | 请愿意作证的同事出具 |
| 公司制度文件 | 证明制度依据 | 截图/拍照员工手册 |
| 绩效考核记录 | 证明绩效情况 | 从系统导出/截图 |
| 培训记录 | 证明公司投入 | 整理培训证书/记录 |
| 获奖/表彰记录 | 证明工作表现 | 整理荣誉证书 |

## 🎬 系统取证指南（互联网公司特有）

### 飞书取证

| 取证内容 | 操作步骤 | 注意事项 |
|---------|---------|---------|
| **聊天记录** | 1. 打开对话<br>2. 手机录屏<br>3. 逐条滚动展示 | 显示时间、对方头像、完整内容 |
| **被踢出群** | 1. 打开群聊<br>2. 录屏展示"你已被移出群聊"<br>3. 记录群名称 | 显示群名称、移出时间 |
| **被禁言** | 1. 尝试发送消息<br>2. 录屏展示禁言提示 | 显示禁言提示内容 |
| **组织架构** | 1. 打开通讯录<br>2. 录屏展示所在部门 | 证明汇报关系 |
| **打卡记录** | 1. 打开考勤<br>2. 录屏展示打卡时间 | 导出近12个月 |

### 钉钉取证

| 取证内容 | 操作步骤 | 注意事项 |
|---------|---------|---------|
| **聊天记录** | 1. 打开对话<br>2. 手机录屏<br>3. 逐条滚动展示 | 显示时间、对方头像 |
| **DING消息** | 1. 打开DING<br>2. 录屏展示已读/未读 | 证明送达 |

## 🎙️ 录音取证规范

### 录音合法性

✅ **合法**：双方对话录音（你参与对话）
❌ **违法**：窃听第三方对话（你未参与）

### 录音操作指南

| 场景 | 录音要点 | 话术示例 |
|------|---------|---------|
| **HR谈话** | 1. 打开手机录音<br>2. 放在桌面/口袋<br>3. 清晰记录对话 | "请问今天是___，您是___吗？公司的决定是___？" |
| **主管谈话** | 1. 确认对方身份<br>2. 记录关键信息<br>3. 结束时确认 | "您是说从___开始让我停岗，工资___，对吗？" |
| **电话沟通** | 1. 开启通话录音<br>2. 确认通话对象<br>3. 记录完整通话 | "您好，请问是___吗？我是___，关于停岗的事..." |

### 录音保存要求

1. **原始载体**：保留录音原始手机，不要删除
2. **备份拷贝**：将录音文件拷贝到电脑/云端
3. **文字整理**：整理录音文字稿，标注关键时间点
4. **时间标注**：文件名包含日期时间，如"20250411_HR谈话_1430"

## 📱 手机截图/录屏规范

### 截图要求

| 内容 | 要求 |
|------|------|
| **完整界面** | 包含顶部状态栏（时间、信号、电量） |
| **清晰可读** | 文字清晰，不模糊 |
| **连续截图** | 长对话分段截图，确保连续性 |
| **原始尺寸** | 不裁剪、不压缩 |

### 录屏要求

| 内容 | 要求 |
|------|------|
| **显示时间** | 顶部状态栏显示当前时间 |
| **网络状态** | 显示网络连接状态（证明在线） |
| **操作清晰** | 手指点击位置可见 |
| **内容完整** | 不遗漏关键信息 |
| **原始文件** | 保存原始录屏文件 |

## ⚖️ 证据在仲裁中的作用

### 举证责任分配

| 待证事实 | 举证责任 | 说明 |
|---------|---------|------|
| **劳动关系存在** | 劳动者 | 提供合同、工资流水等 |
| **工资标准** | 劳动者 | 提供工资条、流水等 |
| **违法解除** | 劳动者 | 提供停岗证据 |
| **解除理由合法** | 用人单位 | 公司需证明解除合法 |
| **工资已支付** | 用人单位 | 公司需提供支付凭证 |

### 常见证据问题

| 问题 | 后果 | 解决方案 |
|------|------|---------|
| 证据原件丢失 | 无法质证 | 提前备份多份 |
| 录音不清晰 | 无法采信 | 检查录音质量 |
| 截图不完整 | 证明力弱 | 确保完整截图 |
| 超过举证期限 | 证据不被采纳 | 及时提交 |
| 证据被篡改 | 可能承担法律责任 | 保留原始文件 |

## 🔒 证据安全提示

1. **立即备份**：收集到的证据当天必须备份
2. **多份保存**：至少保存3份（本地+云端+物理）
3. **防止丢失**：手机不要随意卸载APP，不要恢复出厂设置
4. **防止篡改**：不要编辑、裁剪、修改原始证据
5. **防止泄露**：敏感证据不要随意分享，注意隐私保护
