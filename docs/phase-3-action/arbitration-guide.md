# 劳动仲裁全流程指南

## 全景图

```
发现问题 → 收集证据 → 确定管辖 → 提交申请 → 立案 → 调解 → 开庭 → 裁决 → （不服→起诉）→ 执行
  ↑                                                                ↑
  └─── 1年时效 ───────────────────────────────────────────────────┘
```

## 阶段一：准备期

### 1. 确定管辖（去哪里申请仲裁）
- **选择权在你**：劳动合同履行地（实际工作地）或用人单位所在地
- 两个都能选，选对自己方便的
- 搜索方式：`上海 [区名] 劳动人事争议仲裁委员会`（如：上海浦东新区劳动人事争议仲裁委员会）
- 打电话确认：地址、办公时间、所需材料

### 2. 准备材料清单
| 材料 | 份数 | 说明 |
|------|------|------|
| 仲裁申请书 | N+1份（N=被申请人数量） | 手写或打印，签名按手印 |
| 身份证复印件 | 1份 | 正反面 |
| 劳动合同 | 复印件1份 | 带原件核对 |
| 证据清单 | 1份 | 编号列出，附证明目的 |
| 证据材料 | N+1份 | 按证据清单顺序排列 |
| 用人单位工商信息 | 1份 | 天眼查/企查查打印即可 |

### 3. 仲裁申请书怎么写

**核心要素：**
```
1. 申请人信息：姓名、性别、身份证号、住址、电话
2. 被申请人信息：公司全称、统一社会信用代码、地址、法定代表人、电话
3. 仲裁请求：
   - 请求一：要求被申请人支付 XXX 元（写明计算方式）
   - 请求二：...
   - 每个请求单独列一条，金额明确
4. 事实与理由：
   - 什么时候入职的
   - 做什么工作
   - 发生了什么事
   - 为什么认为公司违法
   - 引用相关法条
5. 此致 上海市[区名]劳动人事争议仲裁委员会（如：上海市浦东新区劳动人事争议仲裁委员会）
```

**常见仲裁请求写法：**
- "要求被申请人支付违法解除劳动合同赔偿金 XXX 元（计算方式：月平均工资 X 元 × 工作年限 X × 2）"
- "要求被申请人支付拖欠工资 XXX 元（X 年 X 月至 X 年 X 月）"
- "要求被申请人支付未签订劳动合同的双倍工资差额 XXX 元"
- "要求被申请人支付加班费 XXX 元"
- "要求被申请人为申请人补缴 X 年 X 月至 X 年 X 月的社会保险"

## 阶段二：申请立案

### 提交方式
1. **现场提交**（推荐）：直接去仲裁委办事窗口
2. **邮寄提交**：EMS 邮寄，保留快递单和签收记录
3. **网上提交**：部分地区支持网上立案（查当地仲裁委官网）

### 立案审查
- 仲裁委收到申请后 **5 个工作日内** 决定是否受理
- 受理的，发给受理通知书
- 不受理的，发给不予受理通知书（拿到这个可以直接去法院起诉）
- 收到受理通知书后，按要求缴纳费用（劳动仲裁 **不收费**）

### 送达
- 仲裁委会把申请书副本送给公司
- 公司有 10 天时间提交答辩书

## 阶段三：调解

### 仲裁前调解（大多数仲裁委都会先安排）
- 仲裁委组织双方调解
- **你可以选择接受或拒绝**
- 调解成功的，出具调解书（有法律效力）
- 调解不成的，进入审理

### 调解策略
- 不要第一个报价，让对方先出
- 知道自己的底线（能接受的最低金额）
- 可以"分期支付"换"总额不降"
- 调解书一旦签了就生效，不能再仲裁同一事项
- 调解金额可以打折，但不要低于应得的 50%

## 阶段四：开庭审理

### 庭前准备
- 熟悉自己的仲裁申请书和证据
- 准备好证据原件（开庭时要核对）
- 列一个发言提纲（我要说什么、重点是什么）
- 准备好公司的可能抗辩和你的反驳

### 开庭流程
```
1. 核对身份 → 双方出示身份证件
2. 宣读权利义务 → 听着就行
3. 申请人陈述 → 按申请书读，补充要点
4. 被申请人答辩 → 听对方说什么
5. 举证质证 → 你出示证据，对方质证；对方出示证据，你质证
6. 辩论 → 双方互相辩论
7. 最后陈述 → 简单总结你的诉求
8. 调解 → 仲裁员会再问是否愿意调解
9. 休庭 → 等裁决
```

### 举证质证技巧
- **质证对方证据的三招**：真实性（是不是真的）、合法性（怎么来的）、关联性（和本案有没有关系）
- 对方拿不出原件的，可以说"对复印件真实性不予认可"
- 对方的证据对你有利的，可以认可

### 开庭注意事项
- 着装整洁，态度端正
- 不要和对方争吵，只和仲裁员说话
- 说事实，不要发泄情绪
- 听不清楚的可以要求重复
- 对方说的不对的地方要明确表示不认可

## 阶段五：裁决

### 裁决时间
- 受理后 **45 天内** 结案
- 案情复杂可延长 **15 天**

### 裁决结果
- 支持你的请求（全部或部分）
- 驳回你的请求
- 超过时限未裁决的，可以向法院起诉

### 终局裁决 vs 非终局裁决
- **终局裁决**：追索劳动报酬不超过当地月最低工资标准 12 个月金额的争议，一裁终局
  - 员工可以起诉，公司不能起诉（只能申请撤销）
- **非终局裁决**：双方都可以在 15 天内向法院起诉

## 阶段六：执行

### 如果公司不履行裁决
1. 裁决书生效后，公司不按期支付的
2. 向 **人民法院** 申请强制执行
3. 申请执行的期限：**2 年**
4. 需要提交：强制执行申请书 + 裁决书原件 + 身份证
5. 法院可以：冻结公司账户、查封财产、限制高消费、纳入失信名单

### 常见问题
- **公司说没钱**：法院会查公司账户，确实没钱可能需要等。但如果公司还在经营，基本都能执行到
- **公司注销了**：追加股东为被执行人（有限责任公司股东有连带责任的情况）
- **公司转移资产**：可以向法院提供线索，法院有权追回

## 时间线汇总

| 阶段 | 时间 | 备注 |
|------|------|------|
| 仲裁时效 | 1年 | 从知道权益受损之日起 |
| 立案审查 | 5个工作日 | 决定是否受理 |
| 答辩期 | 10天 | 公司收到申请书后 |
| 审理期限 | 45天 | 可延长15天 |
| 起诉期限 | 15天 | 收到裁决书后 |
| 申请执行 | 2年 | 裁决生效后 |

## 全流程预估时间
- **最顺利**：调解成功，1-2个月
- **正常情况**：仲裁裁决，3-4个月
- **最久**：仲裁+一审+二审+执行，1-2年
