# 维权文书模板

## 模板一：劳动仲裁申请书

```
劳动人事争议仲裁申请书

申请人：[姓名]，[性别]，[民族]，[出生年月]
身份证号码：[身份证号]
住址：[住址]
电话：[手机号]

被申请人：[公司全称]（务必与营业执照一致）
统一社会信用代码：[信用代码]
住所地：[公司注册地址]
法定代表人：[姓名]，[职务]
电话：[公司电话]

仲裁请求：
一、要求被申请人支付[具体请求，如：违法解除劳动合同赔偿金]人民币[金额]元；
二、要求被申请人支付[第二个请求，如：拖欠工资]人民币[金额]元；
三、要求被申请人为申请人补缴[时间段]的社会保险；
（根据实际情况增减）

事实与理由：

申请人于[入职日期]入职被申请人处，担任[岗位]职务，月平均工资约[金额]元。

[详细描述发生了什么，按时间顺序]

[说明为什么公司的行为违法]

[引用相关法条]

综上所述，被申请人的行为严重侵犯了申请人的合法权益。为维护申请人的合法权益，
特依法提起劳动仲裁，请仲裁委员会依法裁决，支持申请人的全部仲裁请求。

此致

上海市[区名]劳动人事争议仲裁委员会（如：上海市浦东新区劳动人事争议仲裁委员会）

申请人（签名）：
日期：    年  月  日

附件：
1. 申请人身份证复印件
2. 证据清单及证据材料
```

## 模板四：EMS 邮寄说明

仲裁材料如果需要邮寄，使用 EMS（中国邮政特快专递），因为只有 EMS 的送达在法律上被认可。

**邮寄注意事项：**
1. 使用 EMS，不要用其他快递
2. 在快递单"内件品名"处写明：
   - "劳动仲裁申请材料（致XX仲裁委）"
   - 或"解除劳动合同通知书（致XX公司）"
3. 保留快递底单
4. 在 EMS 官网查询签收记录并截图保存
5. 如被退回，不要拆封，原样保存

## 模板五：解除劳动合同通知书（劳动者发出）

**适用场景：** 公司存在违法行为，劳动者主动解除合同并要求经济补偿

```
解除劳动合同通知书

[公司全称]：

本人[姓名]，于[入职日期]入职贵公司，担任[岗位]职务。

因贵公司存在以下违法行为：
1. [具体违法事实，如：未及时足额支付劳动报酬]
2. [具体违法事实，如：未依法为劳动者缴纳社会保险费]

根据《中华人民共和国劳动合同法》第三十八条的规定，本人现正式通知贵公司：
自本通知送达之日起，解除与贵公司的劳动合同。

同时，本人将依法主张经济补偿金及相关合法权益。

请贵公司在解除劳动合同后：
1. 出具解除劳动合同证明书
2. 在15日内办理档案和社会保险关系转移手续
3. 依法支付经济补偿金及相关款项

通知人（签名）：
日期：    年  月  日
```

**法律依据（《劳动合同法》第38条）：**
劳动者可以单方解除合同并主张经济补偿的情形：
- 未提供劳动保护或劳动条件
- 未及时足额支付劳动报酬
- 未依法缴纳社会保险费
- 规章制度违法损害劳动者权益
- 以欺诈、胁迫手段订立合同
- 法律、行政法规规定劳动者可以解除合同的其他情形

**不需要事先告知就可以立即解除的情形：**
- 以暴力、威胁或非法限制人身自由强迫劳动
- 违章指挥、强令冒险作业危及人身安全
