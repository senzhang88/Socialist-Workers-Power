# 上海 Labor Rights Skill - 上海劳动维权全流程导航

一个专为上海地区劳动者设计的AI维权助手技能，帮助员工分析劳动权益受损情况、制定维权计划、计算赔偿金额（含三倍封顶37,000元标准）、指导仲裁申请流程。

## 项目背景

企业有法务团队，而员工只有自己——这个技能的目标就是补上这个信息差。

和普通法律咨询不同，这个技能：
- **站在员工这边** - 不是中立咨询，是帮助维权
- **可执行优先** - 不说"根据第X条规定"，说"你明天该做什么"
- **全流程覆盖** - 从证据收集到仲裁开庭，每一步都有指导
- **开箱即用** - 整合了劳动法规定、地方政策、文书模板，无需联网查询

## 核心场景覆盖

| 维权类型 | 覆盖内容 |
|---------|---------|
| **违法解除** | 无理由辞退、口头辞退 |
| **软裁员/变相辞退** ⭐ | 停岗待业、关闭权限、不让进门但不算开除 |
| 欠薪/克扣工资 | 工资不发、少发、拖欠 |
| 未签劳动合同 | 双倍工资主张 |

| 调岗降薪 | 未经同意调岗应对 |
| 工伤 | 工伤认定与赔偿 |
| 年假/休假 | 未休年假工资计算 |

## 特色功能

### 1. 软裁员应对（你遇到的问题）

公司不书面辞退你，但：
- 口头通知停岗待业
- 关闭门禁、OA权限
- 降薪到基本工资
- 不让进公司但不算开除

**技能提供：**
- 为什么可以主张 2N（违法解除赔偿金）而非只能拿 N
- 如何固定证据（报警、录像、EMS发函）
- 完整时间线和行动清单
- 仲裁申请书写作要点

### 2. 互联网公司常见风险指南 ⭐

针对互联网/科技型企业常见制度风险：
- 严苛旷工认定（迟到1小时算旷工）
- PIP绩效改进计划陷阱
- 薪酬保密高压线
- 加班调休过期陷阱
- 软裁员针对性应对策略
- 制度合法性审查要点

*注：本指南基于行业常见情况总结，不针对任何具体公司*

### 2. 上海地区特殊规定

- 各仲裁委地址、电话
- 三倍封顶标准（2024年：12,434元/月 → **37,302元**封顶）
- 产假天数（158天起）
- 加班费计算基数规则
- 监察投诉不受理时的应对

### 3. 上海社保、公积金政策

- 社保缴费基数上下限（2025-2026年：7,460元 - 37,302元）
- 公积金缴存比例和基数
- 社保/公积金投诉流程
- 失业保险金标准（2,175元/月）
- 工伤保险待遇（2025年最新标准）

### 4. 核心法律法规原文

- 《劳动合同法》核心条文（第38、47、48、87条等）
- 《劳动法》核心条文
- 《劳动争议调解仲裁法》
- 最高人民法院司法解释
- 上海地方性法规

### 5. 实用工具

- 赔偿计算器逻辑（N、2N、双倍工资、加班费）
- 仲裁申请书模板（上海专用）
- 证据清单模板
- EMS邮寄指南

### 3. 赔偿计算器逻辑

支持计算：
- 经济补偿金（N）
- 违法解除赔偿金（2N）
- 未签合同双倍工资
- 加班费
- 未休年假工资
- 工伤赔偿

## 如何使用这个Skill

### 作为Claude Skill使用

1. 将本项目文件夹放到 Claude 技能目录：
   ```bash
   cp -r labor-rights ~/.claude/skills/
   ```

2. 在对话中触发关键词：
   - "我被公司辞退了"
   - "公司欠薪怎么办"
   - "劳动仲裁怎么申请"
   - "软裁员"、"变相辞退"
   - 其他劳动权益相关描述

### 作为项目资料库使用

直接查阅 `references/` 下的各类指南：
- `soft-layoff-guide.md` - 如果你正遭遇软裁员
- `shanghai-regulations.md` - 如果你在上海维权
- `templates.md` - 需要写仲裁申请书或投诉信
- `evidence-checklist.md` - 不知道收集什么证据
