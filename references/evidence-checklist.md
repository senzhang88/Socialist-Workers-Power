# 证据收集指南

## 证据分级

### S 级证据（仲裁庭直接采信）
| 证据 | 获取方式 | 注意事项 |
|------|---------|---------|
| 劳动合同 | 拍照/复印/扫描 | 有公司盖章或签字的原件效力最高 |
| 银行工资流水 | 银行APP导出或柜台打印 | 带银行业务章，至少12个月 |
| 社保缴费记录 | 支付宝/微信市民中心、当地人社APP | 显示缴费基数和月数 |
| 税务完税证明 | 个人所得税APP | 佐证实际收入 |
| 公证书/公证文书 | 公证处 | 对电子证据做公证效力最强 |

### A 级证据（需要与其他证据配合）
| 证据 | 获取方式 | 注意事项 |
|------|---------|---------|
| 工资条 | 拍照/截图 | 显示明细（基本工资、加班费、扣款） |
| 打卡/考勤记录 | 拍照考勤系统页面 | 连续截取，标注日期 |
| 工作群聊天记录 | 录屏+截图 | 录屏比截图更有说服力 |
| 工作邮件 | 转发到个人邮箱 | 保持邮件头完整 |
| 通知书/辞退信 | 拍照原件 | 保留好原件 |
| 录音 | 手机录音 | 不要剪辑，保留原始文件 |

### B 级证据（辅助性）
| 证据 | 获取方式 | 注意事项 |
|------|---------|---------|
| 工牌/门禁卡 | 拍照 | 正反面 |
| 名片 | 拍照 | 有公司名称和职位 |
| 工位照片 | 拍照 | 能看到公司标识 |
| 同事证人证言 | 书面证言+联系方式 | 在职同事作证有难度 |
| 快递/外卖记录 | APP截图 | 佐证加班到很晚 |
| 出差凭证 | 机票/车票/报销单 | 佐证工作事实 |

## 按场景的证据清单

### 场景1：被辞退/违法解除
**必须收集：**
- [ ] 辞退通知书（书面最好，没有的话录音固定）
- [ ] 劳动合同
- [ ] 工资流水（离职前12个月）
- [ ] 工作年限证明（入职时的材料、社保记录）
- [ ] 公司要求你签署的任何文件（不要轻易签字！）

**重要提醒：**
- 口头辞退也要录音确认（"你刚才说的是让我明天不用来了对吧？"）
- 不要主动写"辞职信"或签"自愿离职"——这会直接丧失赔偿金
- 如果公司让你签"协商解除"，看清楚条款再签

### 场景2：欠薪
**必须收集：**
- [ ] 劳动合同（约定工资标准）
- [ ] 银行流水（实际发放记录）
- [ ] 工资条（显示应发 vs 实发）
- [ ] 考勤记录（证明你在工作）
- [ ] 催要工资的聊天记录或录音

### 场景3：加班费
**必须收集：**
- [ ] 考勤/打卡记录（显示加班时间）
- [ ] 工资条（显示没有加班费或加班费不足）
- [ ] 工作安排的聊天记录/邮件（证明是公司安排的加班）
- [ ] 加班审批记录（如果公司有审批制度）
- [ ] 晚间/周末的工作产出物（邮件发送时间、文件修改时间）

### 场景4：未签劳动合同
**必须收集：**
- [ ] 证明劳动关系的所有材料（工牌、工资流水、社保记录、工作群记录）
- [ ] 入职时间的证明
- [ ] 工作内容和受管理的事实证据

### 场景5：工伤
**必须收集：**
- [ ] 事故发生的时间、地点、经过记录（事发当天就写下来）
- [ ] 同事证人信息
- [ ] 监控录像（尽快要求调取，一般保存30天）
- [ ] 就医记录（急诊病历、诊断证明、X光/CT报告）
- [ ] 交通事故认定书（如果是上下班途中）
- [ ] 通知公司工伤的记录（聊天/邮件截图）

## 电子证据保存技巧

### 微信聊天记录
1. 不要删除聊天记录
2. 用录屏功能从头到尾录一遍（不要跳着录）
3. 同时截图关键对话
4. 记录对方的微信号（不是昵称）
5. 如果对方删了消息，录屏时说明"对方撤回/删除了某条消息"

### 钉钉/企业微信
1. 离职后账号会被注销！**在职时就备份**
2. 打卡记录、审批记录、工作日志全部截图
3. 导出个人数据（如果APP支持）

### 邮件
1. 转发到个人邮箱（不要只截图）
2. 保持邮件头完整（发件人、时间、收件人）
3. 如果用的是公司邮箱，离职后可能被关闭，提前转发

### 录音
1. 不要告诉对方你在录音
2. 不要剪辑录音文件
3. 录音中要自然提到：日期、对方身份、关键事实
4. 保存原始文件，备份到云盘
5. 录音示例对话：
   - "王经理，关于你昨天说的让我不用来了这件事..."
   - "那你确认一下，是公司要解除我的劳动合同是吗？"
   - "原因是什么呢？能书面通知我吗？"

## 证据整理建议

仲裁前建议这样整理：
1. 建文件夹，按证据类型分类
2. 打印关键证据 2-3 份（仲裁委一份、自己留一份、给对方一份）
3. 制作证据清单（编号、名称、证明目的）
4. 原件自己保管好，提交复印件
