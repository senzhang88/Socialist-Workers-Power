# 赔偿金额计算指南

## 基础概念

### "月工资"怎么算？
- **离职前12个月的平均应发工资**（税前，含奖金、补贴）
- 不是基本工资，是应发工资总额
- 加班费一般计入平均工资
- 年终奖分摊到各月计算

### "工作年限"怎么算？
- 从入职日到离职日
- 每满1年 = 1个月工资的经济补偿
- 6个月以上不满1年 = 按1年算（1个月工资）
- 不满6个月 = 0.5个月工资

### "当地上年度职工月平均工资"（三倍封顶）
- 高收入者注意：如果月工资超过当地上年度职工月平均工资的 **3倍**
- 按3倍封顶计算
- 年限最高算12年

## 各类赔偿计算

### 一、经济补偿金（N）

**适用场景：** 合法解除但需要补偿的情况
- 协商一致解除（公司提出）
- 无过失辞退（不能胜任工作等）
- 经济性裁员
- 合同到期公司不续签
- 公司破产/吊销执照

**计算公式：**
```
经济补偿金 = 月平均工资 × 工作年限
```

**示例：**
- 月工资 8000，工作 3 年 4 个月
- N = 3.5（3年 + 4个月按0.5算）
- 经济补偿金 = 8000 × 3.5 = 28,000 元

### 二、违法解除赔偿金（2N）

**适用场景：** 公司没有合法理由辞退你
- 口头说"你明天不用来了"
- 找各种借口突然开除
- 用威胁/欺骗手段逼迫离职
- 无事实依据的"严重违纪"

**计算公式：**
```
赔偿金 = 经济补偿金 × 2 = 月平均工资 × 工作年限 × 2
```

**示例：**
- 月工资 12000，工作 2 年
- N = 2
- 2N = 12000 × 2 × 2 = 48,000 元

**选择权：**
- 可以选择拿 2N 赔偿金
- 也可以选择要求继续履行劳动合同（恢复劳动关系 + 补发争议期间的工资）

### 三、未签劳动合同双倍工资

**适用场景：** 入职超过1个月没有签书面劳动合同

**计算方式：**
```
双倍工资差额 = 月工资 × min(未签合同月数, 11)
```
- 从入职第2个月算起
- 最多算11个月（入职满1年后视为已订立无固定期限合同）
- 时效：从应当支付双倍工资之日起算1年

**示例：**
- 入职8个月没签合同，月工资6000
- 双倍工资差额 = 6000 × 7 = 42,000 元（第2到第8个月，共7个月）

### 四、拖欠工资 + 赔偿

**计算方式：**
```
欠薪本金 = 应发工资 - 实发工资
额外赔偿 = 欠薪本金 × 50%~100%（由劳动行政部门责令）
```

**或者按劳动部规定：**
```
额外经济补偿金 = 欠薪本金 × 25%
```

**实操建议：**
- 仲裁时两个都写上，让仲裁员选
- 欠薪的举证责任主要在用人单位

### 五、加班费

**计算公式：**
```
工作日加班费 = 月工资 ÷ 21.75 ÷ 8 × 加班小时数 × 1.5
休息日加班费 = 月工资 ÷ 21.75 ÷ 8 × 加班小时数 × 2
法定假日加班费 = 月工资 ÷ 21.75 ÷ 8 × 加班小时数 × 3
```

**21.75 的由来：**
- 月计薪天数 = (365 - 104) ÷ 12 = 21.75 天

**示例：**
- 月工资 10000，平时每月加班 40 小时
- 小时工资 = 10000 ÷ 21.75 ÷ 8 ≈ 57.47 元
- 每月加班费 = 57.47 × 40 × 1.5 = 3,448.28 元

**注意：**
- 加班费的举证责任：2年内的加班事实由劳动者举证，2年以上的由用人单位举证
- 用人单位自认的加班事实不需要额外举证
- 调休不能替代法定假日加班费

### 六、未休年假工资

**计算公式：**
```
未休年假工资 = 日工资 × 未休天数 × 200%
（已包含正常工作日的工资，所以额外支付200%）
```

**年假天数：**
| 累计工作年限 | 年假天数 |
|-------------|---------|
| 1-10年 | 5天 |
| 10-20年 | 10天 |
| 20年以上 | 15天 |

### 七、工伤赔偿（简化版）

**1-4级伤残（保留劳动关系，退出工作岗位）：**
- 一次性伤残补助金：本人工资 × 21-27个月
- 每月伤残津贴：本人工资 × 75%-90%

**5-6级伤残：**
- 一次性伤残补助金：本人工资 × 14-16个月
- 一次性医疗补助金 + 一次性就业补助金（各省标准不同）

**7-10级伤残：**
- 一次性伤残补助金：本人工资 × 7-13个月
- 解除合同时：一次性医疗补助金 + 一次性就业补助金

**工亡：**
- 丧葬补助金：6个月统筹地区上年度月平均工资
- 供养亲属抚恤金：配偶每月40%，其他亲属每月30%
- 一次性工亡补助金：上年度全国城镇居民人均可支配收入 × 20

## 综合案例

### 案例：被违法辞退 + 欠薪 + 没签合同

**基本情况：**
- 2024年3月入职，2025年1月被口头辞退
- 从未签劳动合同
- 月工资 8000 元（银行转账）
- 2024年12月工资未发
- 在职期间经常加班（平均每月40小时，工作日加班）

**可主张金额：**

| 项目 | 计算 | 金额 |
|------|------|------|
| 违法解除赔偿金(2N) | 8000 × 1(N=10个月≈1) × 2 | 16,000 |
| 未签合同双倍工资 | 8000 × 9（4月到12月） | 72,000 |
| 欠薪 | 8000 × 1 | 8,000 |
| 欠薪额外赔偿 | 8000 × 50%（取较低值） | 4,000 |
| 加班费 | (8000÷21.75÷8) × 40 × 1.5 × 10 | 27,586 |
| **合计** | | **127,586** |

> 注意：实际金额取决于证据充分程度和仲裁庭认定。
