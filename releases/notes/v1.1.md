## ⚡ 一键安装

```bash
claude skills install https://github.com/senzhang88/Socialist-Workers-Power/releases/download/v1.1/work-powers-shanghai-v1.1.skill
```

安装后直接用对话触发：

```
公司让我停岗待业怎么办
帮我算一下违法解除的赔偿
怀孕被辞退，怎么办
试用期被辞了，公司没签合同
```

Skill 自动识别场景 → 检索同类胜诉/败诉判例模式 → 给出"今天/明天做什么 + 证据清单 + 联系方式"。
