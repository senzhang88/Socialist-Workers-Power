# 入职避雷与合同审查指南

> 入职前多花1小时调查，避免未来几个月的麻烦。

## 企业背景调查（入职前必查）

| 查什么 | 怎么查 | 危险信号 |
|--------|--------|---------|
| 经营状态 | 天眼查/企查查 | 注销/吊销/清算 |
| 涉诉信息 | 司法风险→搜"劳动争议" | 大量劳动纠纷 |
| 注册资本 | 看实缴资本 | 认缴高但实缴为0（空壳） |
| 变更记录 | 看变更频率 | 频繁变更法人/股东/地址 |
| 经营异常 | 看经营异常名录 | 地址异常、年报异常 |

## 面试话术识别

| HR说的 | 潜台词 | 风险 |
|--------|--------|------|
| "先干着，合同后面再签" | 可能不签合同 | 极高 |
| "试用期工资打8折，转正后补" | 转正可能不补 | 高 |
| "工资发现金，避税" | 社保可能按最低基数缴 | 高 |
| "竞业限制是标准条款" | 离职后可能受限 | 高 |

## 合同必查条款

### 薪资
- 薪资构成必须写明（基本工资+绩效+补贴）
- 试用期工资不得低于转正工资的80%
- 发薪日必须明确到具体日期

### 试用期法定上限

| 合同期限 | 试用期上限 |
|---------|-----------|
| 3个月-1年 | 1个月 |
| 1年-3年 | 2个月 |
| 3年以上/无固定期限 | 6个月 |

注意：同一单位只能约定一次试用期。

### 竞业限制（高风险）
- 限制期限最长2年
- 补偿金标准：不低于离职前12个月平均工资的30%
- 上海标准：低于最低工资的按最低工资付
- **公司不付补偿金超过3个月，劳动者可解除竞业限制**

### 看到这些条款要拒绝
- "自愿放弃社保" → 违法无效
- "工伤自负" → 违法无效
- "加班不给加班费" → 违法
- "入职押金/服装费" → 违法
- "离职需提前3个月申请" → 试用期3天/转正30天

## 入职第一周必须做的事

1. **要求合同一式两份**，自己保留原件
2. **拍照/扫描备份**所有签署文件
3. **确认**：工资发放日、社保基数、年假天数、加班审批流程
4. **保存证据**：offer邮件、员工手册签收页、第一天考勤记录

## 入职后发现被骗

| 情况 | 应对 |
|------|------|
| 实际薪资与offer不符 | 书面提出异议，保留offer和合同 |
| 超过1个月不签合同 | 可主张双倍工资（最多11个月） |
| 超过1年不签合同 | 视为无固定期限合同 |
| 不缴社保 | 投诉12333，要求补缴 |
