# 劳动仲裁全流程指南

> 仲裁是维权的核心途径。本指南告诉你从准备到开庭的每一步怎么做。

## 🎯 仲裁前确认

### 判断标准

**✅ 可以申请仲裁：**
- 被违法辞退（无理由/程序违法）
- 被拖欠工资
- 未缴/少缴社保（但社保补缴建议走监察更快）
- 未休年假不给补偿
- 加班费不给
- 其他劳动权益受损

**❌ 仲裁不受理：**
- 超过1年仲裁时效（从离职或权益受侵害之日起）
- 已申请仲裁且正在处理中（一事不再理）
- 已经法院起诉

### 管辖确定

**去哪里申请？**
- **劳动合同履行地**（实际工作地）→ 推荐
- **用人单位所在地**（公司注册地）

**建议：** 选择实际工作地，方便取证和出庭。

**上海各仲裁委地址电话：** 见 `references/shanghai.md`

### 必备材料（缺一不可）

```

身份证号码：XXXXXXXXXXXXXXXXXX
住址：XX市XX区XX路XX号
联系电话：138XXXXXXXX

被申请人：XX公司
统一社会信用代码：XXXXXXXXXXXXXX（查工商信息）
住所地：XX市XX区XX路XX号（注册地址）
法定代表人：XXX
联系电话：021-XXXXXXXX

仲裁请求：
1. 请求确认被申请人于2025年X月X日违法解除与申请人的劳动合同；
2. 请求裁决被申请人支付违法解除劳动合同赔偿金人民币XX元（37,302元×X年×2）；
3. 请求裁决被申请人支付2025年X月至X月工资差额人民币XX元；
4. 请求裁决被申请人支付未休年假工资人民币XX元；
5. 请求裁决被申请人出具离职证明并办理社保公积金转移手续。

事实与理由：

申请人于2020年X月X日入职被申请人公司，担任XX职务，双方签订书面劳动合同。

2025年X月X日，被申请人在未与申请人协商一致的情况下，口头通知申请人停岗待业，
并将申请人工资从XX元/月降至XX元/月。同时，被申请人关闭了申请人的门禁权限
和OA系统权限，导致申请人无法正常提供劳动。

被申请人的上述行为已构成《劳动合同法》第38条规定的"未按照劳动合同约定
提供劳动条件"以及"未及时足额支付劳动报酬"。

被申请人通过上述手段实质上解除了与申请人的劳动合同，但拒不出具书面解除通知，
试图规避支付经济补偿金的法律责任。被申请人的行为已构成违法解除劳动合同，
申请人依据《劳动合同法》第48条、第87条之规定，主张违法解除赔偿金（2N）。

此外，被申请人还应支付申请人工资差额、未休年假工资，并出具离职证明。

综上所述，被申请人的行为严重侵害了申请人的合法权益，恳请贵委依法裁决，
支持申请人的全部仲裁请求。

此致

上海市XX区劳动人事争议仲裁委员会

申请人：XXX（签名）
日期：2025年X月X日
```

**撰写要点：**
- 时间线清晰（入职→事件发生→离职/停岗）
- 明确法律依据（劳动合同法第X条）
- 仲裁请求具体（金额明确）
- 事实陈述客观（不要情绪化）

## 🏢 提交申请

### 现场提交流程

1. **取号排队**
   - 到仲裁委立案大厅取号
   - 说明是"申请劳动仲裁"

2. **窗口审核**
   - 提交申请书和材料
   - 工作人员审核
   - 可能需要补充或修改

3. **获得受理**
   - 审核通过后，缴纳仲裁费
   - **劳动仲裁免费！**
   - 领取《受理通知书》
   - 记住案件编号，用于查询进度

### 材料审核常见问题

| 问题 | 解决方式 |
|------|---------|
| 申请书格式不对 | 按窗口提供的模板修改 |
| 证据材料不全 | 回去补充后再来 |
| 管辖错误 | 去正确的仲裁委 |
| 被申请人信息不全 | 补充工商信息 |

## ⏱️ 仲裁流程时间线

```
提交申请
    ↓ （5-10个工作日）
立案受理 ← 拿到《受理通知书》
    ↓ （5日内送达被申请人）
对方收到申请书
    ↓ （10日内提交答辩书）
对方答辩 ← 你会收到答辩书副本
    ↓ （1-3个月）
开庭通知 ← 提前5日通知
    ↓
开庭审理 ← 带齐证据原件
    ↓ （数日至1个月）
出裁决 ← 领取《仲裁裁决书》
    ↓ （15日内）
不服可起诉 ← 去基层法院
```

**总体时间：** 2-6个月（案情复杂可能更长）

## ⚖️ 开庭审理

### 开庭准备

**携带物品：**
- 身份证原件
- 所有证据原件
- 仲裁申请书副本
- 笔和纸（记录）

**着装：** 整洁得体，不要拖鞋背心

### 庭审流程

1. **核对身份** - 出示身份证
2. **宣布纪律** - 听仲裁员宣读
3. **申请人陈述** - 读仲裁请求和事实理由（或简要陈述）
4. **被申请人答辩** - 对方陈述
5. **举证质证** - 出示证据，对方质证
6. **辩论** - 双方辩论
7. **最后陈述** - 简要总结
8. **调解** - 仲裁员询问是否愿意调解
9. **签署庭审笔录** - 核对后签字

### 举证质证技巧

**举证（出示证据）：**
- 按证据清单顺序出示
- 说明每份证据证明什么
- 原件给对方看，自己保留

**质证（反驳对方证据）：**
- 对对方证据的真实性、合法性、关联性发表意见
- "对证据1的真实性认可，但证明目的不认可"
- "对证据2的合法性不认可，系非法取证"

**常见质证话术：**
- "该证据没有原件，对真实性不认可"
- "该证据与本案无关"
- "该证据系事后伪造"

### 辩论要点

**围绕争议焦点：**
- 劳动关系是否解除？
- 解除是否合法？
- 赔偿金计算标准？

**引用法律依据：**
- "根据《劳动合同法》第XX条规定..."
- "根据《最高人民法院关于XX的解释》..."

**不要说：**
- 情绪化的话（"公司太黑了"）
- 与案件无关的话
- 猜测性的话（"公司肯定是故意的"）

## 🏆 裁决与后续

### 拿到裁决书后

**胜诉（支持你的请求）：**
- 公司15日内不起诉 → 裁决生效
- 公司不履行 → 申请法院强制执行

**败诉（驳回请求）：**
- 15日内可以向基层法院起诉
- 准备上诉材料

**部分胜诉：**
- 对不服的部分可以起诉

### 强制执行

**申请条件：**
- 裁决已生效（15日已过，双方未起诉）
- 公司未履行

**申请方式：**
- 向被执行人住所地或财产所在地基层法院申请
- 提交《强制执行申请书》+ 裁决书 + 身份证明

## 💡 实战技巧

### 提高胜率的关键

1. **证据充分** - 证据决定胜负
2. **法律依据准确** - 引用正确的法条
3. **时间线清晰** - 让仲裁员容易理解
4. **态度专业** - 不情绪化，讲道理
5. **考虑调解** - 有时调解比裁决更快拿到钱

### 常见误区

❌ **误区1：仲裁必须请律师**
> 事实：劳动仲裁可以自辩，简单案件自己处理即可

❌ **误区2：证据越多越好**
> 事实：证据要精不要多，每份证据都要有证明目的

❌ **误区3：开庭时说越多越好**
> 事实：言多必失，围绕焦点说，不要啰嗦

❌ **误区4：裁决后就完事了**
> 事实：公司可能起诉或拖延执行，要做好持久战准备
