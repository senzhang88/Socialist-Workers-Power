# 仲裁开庭指南

> 开庭不可怕，劳动仲裁更像调解，不像电视剧里的法庭。

## 庭前准备

### 一周前
- 熟读仲裁申请书，每个请求都能说清来龙去脉
- 整理证据原件，按证据清单顺序排列
- 打印证据材料2-3套（仲裁庭/对方/自留各一套）
- 准备开庭发言提纲（3-5分钟能说完）
- 想好公司可能的抗辩理由和你的反驳

### 当天
- 提前15-30分钟到达
- 带齐：身份证原件、证据原件、申请书副本、纸笔
- 着装整洁正式，关手机静音

## 开庭流程

### 1. 核对身份（5分钟）
出示身份证，确认对方出庭人员身份。

### 2. 申请人陈述（5-10分钟）
你的主场，按这个结构说：
1. **一句话概括**：入职时间、岗位、公司做了什么
2. **按时间线讲事实**：不要跳来跳去
3. **说明违法之处**：违反了哪条规定
4. **重申请求**：要求支付XXX元

**技巧**：声音洪亮、说事实不说评价、每个关键事实说"详见证据X"、只看仲裁员。

### 3. 被申请人答辩（5-10分钟）
认真听，记笔记。他们否认了什么？理由是什么？有什么漏洞？

### 4. 举证质证（最关键，20-40分钟）

**你举证**：逐一说明"证据X，证明XX事实"

**你质证对方证据**，从三个角度：
1. **真实性**："这是复印件，我方要求查看原件"
2. **合法性**："这是公司单方制作，未经我方确认"
3. **关联性**："这份证据和我主张的事实没有关系"

**质证金句**：
- "对这份证据的真实性不予认可"
- "该证据系被申请人单方制作，未经我方确认"
- "对方提供的证据恰恰说明..."

### 5. 辩论（10-20分钟）
围绕争议焦点辩论。抓住对方漏洞、用证据说话、不被对方情绪带跑。

### 6. 最后陈述（2-3分钟）
简短总结："请仲裁庭支持我方全部仲裁请求。"

### 7. 再次调解
仲裁员问是否调解。可以同意但要有底线，也可以直接拒绝。

## 常见公司抗辩及应对

| 公司抗辩 | 你的应对 |
|---------|---------|
| "他是主动辞职的" | 要求出示辞职信原件；用录音/聊天记录证明是被辞退 |
| "严重违反规章制度" | 要求出示制度且证明你签字确认过；要求举证"严重"标准 |
| "经营困难" | 经营困难不是违法解除理由；合法裁员需走法定程序 |
| "绩效不达标" | 要求出示考核标准和签字确认的考核结果；不胜任需先培训/调岗 |
| "签了协商解除协议" | 检查是否胁迫/欺骗签订；内容是否显失公平 |
| "考勤记录没有了" | 考勤记录由用人单位保管，举证责任在公司 |
| "他不是我们的员工" | 出示工资流水、社保记录、工牌、工作群记录 |

## 心态管理

- 紧张是正常的，仲裁员不是法官，更像调解
- 说错了可以更正，不要慌
- 不要和对方吵架，只和仲裁员沟通
- 没说完的可以提交书面补充意见

## 庭后

- 裁决时间：一般45天内
- 可提交"代理词"补充观点
- 收到裁决书后15天内决定是否起诉
- 对裁决不服可向法院起诉
