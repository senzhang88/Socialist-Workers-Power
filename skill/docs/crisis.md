# 软裁员/变相辞退应对指南

> 公司不直接辞退你，但用停岗、降薪、关闭权限等手段逼你走。这是典型的"软裁员"。

## 🚨 识别软裁员

**常见套路：**

| 公司行为 | 目的 | 你的应对 |
|---------|------|---------|
| 口头通知停岗待业，降薪 | 逼你受不了自己走 | 每天去公司打卡，被拦就报警 |
| 关闭门禁、OA权限 | 让你无法工作取证 | 拍照录像，EMS发函要求恢复 |
| 不安排工作，晾着你 | 耗时间让你辞职 | 书面要求安排工作，保留证据 |
| 调岗到偏远地点 | 增加通勤成本 | 书面拒绝，要求按原合同履行 |
| 加大工作量/恶意考核 | 让你完不成任务 | 保留工作记录，证明自己完成了 |

**核心原则：** 公司想让你主动走，这样不用付赔偿。你不要上当！

---

## ⚖️ 法律定性：公司违法了

公司行为违反《劳动合同法》：

| 公司行为 | 违反的法条 | 你能主张 |
|---------|-----------|---------|
| 单方面降薪 | 第35条（变更合同需协商一致） | 按原工资补发差额 |
| 关闭门禁/权限 | 第38条（未提供劳动条件） | 2N赔偿金 |
| 不让进门 | 同上 | 2N赔偿金 |
| 口头辞退无书面 | 第50条（解除需书面通知） | 2N赔偿金 |

**关键：** 公司违法在先，你可以主张**2N违法解除赔偿金**，不是被迫解除N！

## 🎯 应对策略（主张2N）

### 第一步：固定证据（72小时内）

**核心证据清单：**

| 证据 | 如何固定 | 重要程度 |
|------|---------|---------|
| **无法进公司** | 报警拿报警回执，全程录像 | ⭐⭐⭐ 最关键 |
| **门禁被取消** | 刷卡失败照片/视频 | ⭐⭐⭐ |
| **OA被关闭** | 登录失败截图 | ⭐⭐⭐ |
| **降薪证据** | 工资流水对比，HR录音 | ⭐⭐ |
| **停岗通知** | 微信/邮件记录，录音 | ⭐⭐ |

**证据获取方式：**

1. **报警取证**（最重要）
   - 去公司被拦→立即报警
   - 警察到场后说明情况
   - 要求出具报警回执或登记记录
   - 自己全程录像

2. **EMS发函**
   - 发《要求恢复劳动条件通知书》
   - 快递单写清文件名称
   - 保留底单和签收记录

3. **系统截图**
   - OA登录失败页面
   - 门禁刷卡失败提示
   - 工作群被踢出记录

### 第二步：绝对不做的事

❌ **不要主动说"我辞职"**（微信/邮件/口头都不要）
❌ **不要签自愿离职协议**
❌ **不要发第38条被迫解除**（只能拿N，错过2N）
❌ **不要不去公司**（会被算旷工）

✅ **要做的事：**
- 每天去公司打卡（被拦是证据）
- 每次被拦都报警
- 给公司发书面通知要求恢复条件
- 准备仲裁材料

### 第三步：书面通知公司

**《要求恢复劳动条件通知书》模板：**

```
致：XX公司

本人于X年X月X日入职公司。X年X月X日，公司在未协商一致的情况下，
口头通知本人停岗待业，关闭本人门禁权限和OA权限，导致无法正常提供劳动。

上述行为已严重违反《劳动合同法》相关规定。

现正式通知公司：
1. 请在收到本通知后3个工作日内恢复本人正常工作岗位；
2. 请按原工资标准补发停岗期间工资差额；
3. 如公司拒不改正，本人将依法通过劳动仲裁维护权益。

特此通知。

通知人：XXX
日期：XXXX年XX月XX日
```

**发送方式：**
- EMS邮寄（快递单注明"要求恢复劳动条件通知书"）
- 同时发邮件/微信（如果还能发）
- 保留快递底单+签收记录

### 第四步：申请劳动仲裁（主张2N）

**仲裁请求：**
```
1. 确认公司于X年X月X日违法解除与申请人的劳动合同
2. 请求裁决公司支付违法解除劳动合同赔偿金37,302元×工作年限×2
3. 请求裁决公司支付X年X月至X年X月工资差额
4. 请求出具离职证明并办理社保转移
```

**关键要点：**
- 公司没有书面解除，但行为已实质解除劳动关系
- 引用《劳动合同法》第48条、第87条
- 主张2N而不是N

## 📋 行动清单（按天）

### 第1天（今天）
- [ ] 去公司打卡（即使进不去）
- [ ] 被拦立即报警，拿报警回执
- [ ] 拍摄门禁/OA失效证据
- [ ] 整理劳动合同、工资流水

### 第2-3天
- [ ] EMS发《要求恢复劳动条件通知书》
- [ ] 继续每天去公司打卡
- [ ] 每次被拦都报警
- [ ] 收集所有证据备份

### 第4-7天
- [ ] 公司无回应或拒绝→准备仲裁材料
- [ ] 写仲裁申请书
- [ ] 准备证据清单
- [ ] 查询企业工商信息

### 第8-14天
- [ ] 提交仲裁申请
- [ ] 或先发律师函施压

## ⚠️ 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 签自愿离职 | 一分钱赔偿拿不到 | 任何文件都不签，或注明"不代表认可" |
| 主动发第38条解除 | 只能拿N，错过2N | 先主张2N，实在不行再考虑N |
| 不去公司 | 被算旷工，公司合法辞退 | 每天去，被拦是证据 |
| 不发书面通知 | 公司否认不让进门 | 必须EMS发函，保留送达证据 |
| 只截图不报警 | 证据效力弱 | 报警+录像，第三方固定证据 |

## 💡 关键心态

**软裁员是心理战。** 公司想让你焦虑、自我怀疑、最终认输走人。

**你的优势：**
1. 法律站在你这边（公司确实违法了）
2. 你有时间（只要你不辞职，公司要么恢复条件，要么构成违法解除）
3. 你可以主动（决定继续耗还是发起仲裁）

**不要做的事：**
- 不要在社交媒体发泄（可能被截图）
- 不要和同事说太多（可能传到公司）
- 不要冲动辞职（正中公司下怀）
