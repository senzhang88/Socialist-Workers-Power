## 触发场景

当用户提到以下情况时，本技能会自动触发：
- 被辞退/裁员/软裁员（停岗、降薪、关闭权限）
- 劳动纠纷/违法解除劳动合同/被迫离职
- 欠薪/社保未缴/加班费争议
- 需要申请劳动仲裁/计算赔偿金额（N+1/2N）
- 证据收集/录音取证/劳动监察投诉
- 试用期被辞/调岗降薪/不让进门

## 使用指南

### 作为用户

直接向Claude描述你的劳动纠纷情况，例如：
- "公司让我停岗待业，工资降到基本工资，门禁权限也关了"
- "帮我算一下赔偿，我月薪25000，工作了5年半"
- "我要申请劳动仲裁，需要准备什么材料？"

### 作为开发者

#### 运行评估

```bash
# 进入技能目录
cd ~/.claude/skills/labor-rights-shanghai

# 聚合评估结果
python scripts/aggregate_benchmark.py 'logs/iteration-1/*.json'

# 生成HTML报告
python eval-viewer/generate_review.py results.json -o report.html
```

#### 优化迭代流程

1. **修改SKILL.md** - 调整响应策略、模板或数据
2. **运行评估** - 使用evals.json中的测试用例
3. **分析结果** - 查看聚合报告和可视化页面
4. **记录变更** - 在ITERATION_LOG.md中记录迭代

## 评估维度

- **可执行性**: 是否给出明确行动步骤（今天做什么、明天做什么）
- **准确性**: 上海数据是否正确（社平12,434、三倍封顶37,302等）
- **完整性**: 是否覆盖证据/时效/联系方式等所有关键点
- **立场**: 是否明确站在员工侧，给出维权策略
- **工具使用**: 是否正确调用赔偿计算器或引用文件
