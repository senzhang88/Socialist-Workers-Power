# 赔偿金额计算指南

> 手把手教你计算经济补偿金、赔偿金、加班费、年假补偿。

## 💰 核心概念

### "N"是什么？

**N = 经济补偿金**（合法解除时支付）

计算方式：
```
N = 月平均工资 × 工作年限
```

**工作年限计算：**
- 每满1年 = 1年
- 6个月以上不满1年 = 1年  
- 不满6个月 = 0.5年

**示例：**
- 工作3年2个月 = 3.5年
- 工作5年7个月 = 6年

---

### "2N"是什么？

**2N = 违法解除赔偿金**（公司违法解除时支付）

计算方式：
```
2N = N × 2 = 月平均工资 × 工作年限 × 2
```

**适用情形：**
- 无理由辞退
- 口头辞退无书面通知
- 停岗降薪关闭权限（软裁员）
- 其他违法解除情形

---

### "N+1"是什么？

**N+1 = 经济补偿金 + 代通知金**

**适用情形：**
- 公司依据《劳动合同法》第40条解除（不能胜任、医疗期满、客观情况变化）
- 公司未提前30日通知，需支付代通知金（1个月工资）

**计算方式：**
```
N+1 = 月平均工资 × 工作年限 + 月平均工资
     = 月平均工资 × (工作年限 + 1)
```

---

## 📊 月平均工资怎么算？

### 计算基数

**公式：**
```
月平均工资 = 离职前12个月工资总额 ÷ 12
```

**包括：**
- ✅ 基本工资
- ✅ 绩效工资
- ✅ 奖金（年终奖÷12计入）
- ✅ 补贴（交通、餐补等固定发放）
- ✅ 加班费

**不包括：**
- ❌ 报销费用
- ❌ 一次性奖励（非常规）
- ❌ 股权/期权收益

**注意：** 是**税前**工资，不是到手工资！

## 🏙️ 上海三倍封顶（高收入者注意）

### 封顶规则

**上海2024年度标准（2025.07-2026.06）：**

| 项目 | 数值 |
|------|------|
| 社平工资 | 12,434元/月 |
| **三倍封顶** | **37,302元/月** |

**规则：**
- 月薪超过37,302元的，按37,302元计算
- 工作年限最高计算12年

**示例：**

**案例1：月薪50,000元，工作5年**
```
封顶月工资 = 37,302元（不是50,000元）
工作年限 = 5年（不满12年，正常计算）

N = 37,302 × 5 = 186,510元
2N = 186,510 × 2 = 373,020元
```

**案例2：月薪50,000元，工作15年**
```
封顶月工资 = 37,302元
工作年限 = 12年（超过12年按12年算）

N = 37,302 × 12 = 447,624元
2N = 447,624 × 2 = 895,248元
```

## 💵 其他补偿项目

### 加班费

**计算公式：**
```
小时工资 = 月平均工资 ÷ 21.75 ÷ 8

工作日加班费 = 小时工资 × 1.5 × 加班小时
休息日加班费 = 小时工资 × 2 × 加班小时
法定假日加班费 = 小时工资 × 3 × 加班小时
```

**示例：**
月薪8,000元，小时工资 = 8000 ÷ 21.75 ÷ 8 = 45.98元
- 工作日加班10小时：45.98 × 1.5 × 10 = 689.7元
- 休息日加班8小时：45.98 × 2 × 8 = 735.68元
- 法定假日加班8小时：45.98 × 3 × 8 = 1103.52元

### 未休年假补偿

**计算公式：**
```
日工资 = 月平均工资 ÷ 21.75
年假补偿 = 日工资 × 未休天数 × 3
```

**示例：**
月薪8,000元，未休年假5天
- 日工资 = 8000 ÷ 21.75 = 367.82元
- 补偿 = 367.82 × 5 × 3 = 5,517.3元

### 欠薪加付赔偿金

**计算公式：**
```
加付赔偿金 = 欠薪金额 × 50%至100%
```

**说明：**
- 需先经劳动行政部门责令支付
- 逾期不支付才产生加付赔偿金
- 实际操作中可主张25%额外经济补偿（劳动部规定）

## 🎯 完整案例计算

### 案例1：普通收入，违法解除

**基本情况：**
- 工作年限：3年5个月 = 3.5年
- 月薪：8,000元
- 解除情形：公司口头辞退，无书面通知（违法解除）
- 未休年假：5天

**计算：**
```
违法解除赔偿金（2N）
= 8,000 × 3.5 × 2
= 56,000元

未休年假补偿
= (8000 ÷ 21.75) × 5 × 3
= 5,517元

合计：61,517元
```

### 案例2：高收入，违法解除

**基本情况：**
- 工作年限：5年
- 月薪：50,000元（超过37,302元封顶）
- 解除情形：软裁员（停岗+降薪+关闭权限）

**计算：**
```
违法解除赔偿金（2N）
= 37,302 × 5 × 2
= 373,020元

（月薪超过封顶，按封顶计算）
```

### 案例3：合法解除（N+1）

**基本情况：**
- 工作年限：2年3个月 = 2.5年
- 月薪：10,000元
- 解除情形：公司提前30日通知解除（客观情况变化）

**计算：**
```
经济补偿（N）
= 10,000 × 2.5
= 25,000元

（公司已提前30日通知，无需+1）

如未提前通知：
N+1 = 10,000 × (2.5 + 1) = 35,000元
```

## ⚠️ 常见误区

### 误区1：月工资按基本工资算
> **错误！** 应按离职前12个月平均应发工资（税前，含奖金补贴）。

### 误区2：年终奖不计入平均工资
> **错误！** 年终奖应分摊到对应月份计入平均工资。

### 误区3：加班费不计入平均工资
> **错误！** 加班费应计入平均工资。

### 误区4：超过封顶的年限也按实际算
> **错误！** 高收入者（月薪>3倍社平）工作年限最高12年。

### 误区5：赔偿金还要扣税
> **注意！** 经济补偿金在当地社平工资3倍以内免征个税，超过部分需缴税。
