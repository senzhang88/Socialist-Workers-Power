## 💰 赔偿计算速查

### 计算公式

```
N（经济补偿金）= 月平均工资 × 工作年限
2N（违法解除赔偿金）= N × 2 = 月平均工资 × 工作年限 × 2

工作年限计算：
- 每满1年 = 1年
- 6个月以上不满1年 = 1年
- 不满6个月 = 0.5年

月平均工资：
- 离职前12个月平均工资（税前，含奖金补贴）
- 超过37,302元按37,302元计算（三倍封顶）
- 高收入者年限最高12年
```

### 速查表

| 工作年限 | 月薪10,000元 (N) | 月薪50,000元 (2N封顶) |
|---------|------------------|----------------------|
| 1年 | 10,000元 | 74,604元 |
| 2年 | 20,000元 | 149,208元 |
| 3年 | 30,000元 | 223,812元 |
| 5年 | 50,000元 | 373,020元 |
| 10年 | 100,000元 | 745,040元 (37,302×10×2) |

### 其他补偿

| 项目 | 计算方式 |
|------|---------|
| **加班费** | 工作日1.5倍 / 休息日2倍 / 法定假日3倍 |
| **未休年假** | 日工资 × 未休天数 × 3 |
| **欠薪** | 本金 + 25%-100%赔偿金 |

## 📋 仲裁流程时间线

| 阶段 | 时间 | 用户行动 |
|------|------|---------|
| **立案** | 提交申请后5-10个工作日 | 等通知，可电话查询 |
| **送达** | 立案后10日内 | 公司收到申请书 |
| **答辩期** | 收到后10日 | 公司提交答辩 |
| **开庭** | 立案后1-3个月 | 准备证据、出庭 |
| **裁决** | 开庭后数日至1个月 | 等裁决书 |
| **起诉** | 收到裁决后15日内 | 不服可起诉 |

## 🚨 紧急行动清单

### 被口头辞退当天
- [ ] 要求书面通知（HR不给也要提出）
- [ ] 拍照/录像工作场所状态
- [ ] 保存所有工作群聊天记录
- [ ] 不要签任何"自愿离职"文件

### 被停岗/关闭权限当天
- [ ] 去公司打卡（即使进不去）
- [ ] 被拦就报警（拿报警回执）
- [ ] EMS发《要求恢复劳动条件通知书》
- [ ] 截图保存OA/邮箱被关闭的证据

### 准备仲裁前
- [ ] 整理12个月工资流水
- [ ] 收集劳动合同、考勤记录
- [ ] 下载企业工商信息
- [ ] 写仲裁申请书（3份）
- [ ] 确定管辖仲裁委

## ⚠️ 重要提醒

1. **仲裁时效1年** - 从离职之日起算，不要拖！
2. **劳动仲裁免费** - 不需要交任何费用
3. ** EMS邮寄重要文件** - 保留底单和签收记录
4. **不要签空白文件** - 任何签字前仔细阅读
5. **录音取证合法** - 你是对话参与方即可录音
6. **三倍封顶37,302元** - 高收入者赔偿上限
7. **先监察后仲裁** - 欠薪可走监察（快），赔偿要走仲裁
