## 一、违法解除（最高频，胜率约 60%）

📖 完整 by-type 案例：<https://z.dasszss.com/references/cases/by-type/违法解除/>

### ✅ 胜诉模式

| 高发胜诉关键 | 法院通常怎么判 |
|--------------|----------------|
| **公司无法提供解除理由的书面证据** | 判违法解除，2N 赔偿金 |
| **解除依据"严重违纪"但规章制度未经民主程序公示** | 制度无效，判违法解除 |
| **以"末位淘汰""不胜任工作"解除但未经培训或调岗** | 直接判违法（参照指导案例 183 号） |
| **解除时正处医疗期 / 三期** | 触发《劳动合同法》第 42 条限制 → 违法 |
| **协商解除签字带胁迫情节（如威胁拉黑、不支付应得款项）** | 协议可被撤销，回到违法解除认定 |

### ❌ 败诉模式（重点警示）

| 高发败诉原因 | 怎么避免 |
|--------------|----------|
| **劳动者拿不出实际工资证据**（只有合同基本工资，奖金/加班费打卡截图缺失） | 提前 12 个月留存：工资条 / 银行流水 / 个税 APP 月度收入 |
| **签了"自愿离职"或"协商一致"协议** | 任何离职文件**先拍照再说"我看一下"**，不在现场签字 |
| **公司用充足的"考核记录"证明不胜任** | 在职时定期保存自己的产出/绩效自评/邮件确认 |
| **超过仲裁时效 1 年才提起** | 离职日开始倒计 365 天，不要拖 |
| **没有书面解除通知，公司辩称"是你自己不来上班"** | 关键时点 EMS 邮寄《要求恢复劳动条件通知书》 |

### 🎯 关键证据清单

1. 劳动合同（含历次变更）
2. 12 个月工资银行流水（税前）
3. 工作交接、办公环境、门禁日志的影像证据
4. 公司发出的解除通知 / 微信通知 / 邮件
5. 同事证言（理想但难拿）
6. 个人 OA / 邮箱被关闭截图（**关键时点**）

## 二、软裁员·不提供劳动条件（互联网公司高发）

📖 完整 by-type 案例：<https://z.dasszss.com/references/cases/by-type/软裁员-不提供劳动条件/>

### ✅ 胜诉模式

| 高发胜诉关键 | 法院判定逻辑 |
|--------------|--------------|
| **公司单方关闭门禁 / OA / 邮箱权限** | 构成《劳动合同法》第 38 条"未提供劳动条件" |
| **停岗后无任何工作安排，工资降到基本工资** | 同上 + 工资差额可主张 |
| **被强行调岗到无意义岗位（如仓管、扫地）** | 构成实质性变更，违反第 35 条 |
| **EMS 邮寄过《要求恢复劳动条件通知书》而公司不回应** | 程序到位，进可主张 38 条被迫解除 + N 补偿，退可"等公司动手"主张违法解除 2N |

### ❌ 败诉模式

| 失败原因 | 应对 |
|----------|------|
| **劳动者主动签了"协商一致解除"** | **不要签**，停岗期间也是劳动关系存续 |
| **没有"权限被关闭"的截图证据** | 事发当天就要截图（OA、邮箱、企业微信、门禁拒绝） |
| **直接旷工不去打卡，公司以"严重违纪"反辞** | 即使进不去也要去打卡，被拦就报警拿回执 |

### 🎯 软裁员 4 步行动法

```
第 1 步（事发当天）：
  - 截图所有被关闭的权限
  - 去公司打卡（进不去就报警拿回执）
  - 不签任何文件

第 2 步（一周内）：
  - EMS 寄《要求恢复劳动条件通知书》
  - 整理 12 个月工资流水
  - 录音保存关键对话

第 3 步（半月-1月）：
  - 等公司反应：恢复 / 强行解除 / 继续耗
  - 如公司继续耗 → 第 38 条被迫解除 + N
  - 如公司发解除通知 → 第 87 条违法解除 + 2N

第 4 步：仲裁立案
  - 立案地：实际工作地或公司注册地仲裁委
  - 仲裁请求金额必须明确
```

## 三、年终奖争议（金融/互联网高发）

📖 完整 by-type 案例：<https://z.dasszss.com/references/cases/by-type/年终奖/>

### ✅ 胜诉模式

| 关键事实 | 判决倾向 |
|----------|----------|
| **合同 / offer 明确写"按 X 个月发放年终奖"** | 必发，离职前在岗满整年的全额，跨年的按比例 |
| **历年实际发放（连续 2-3 年）** | 视为劳动报酬，构成劳动者合理预期 |
| **公司因争议恶意阻却条件成就**（用《民法典》第 159 条） | 视为条件已成就，仍应发放 |

### ❌ 败诉模式

| 失败原因 |
|----------|
| **合同/制度明确写"年终奖系公司单方福利，离职无权要求"** |
| **入职不满整年且公司从未发过年终奖** |
| **离职前未提及年终奖、离职后才主张** |

### 🎯 关键法条

- 《民法典》第 159 条：**当事人为自己的利益不正当地阻止条件成就的，视为条件已经成就**（最强武器，对应 NetEase / 滴滴等大厂"离职即作废"条款）
- 上海高院倾向：年终奖**不是单方福利**而是劳动报酬

## 四、医疗期 / 三期女职工保护

📖 完整 by-type 案例：<https://z.dasszss.com/references/cases/by-type/医疗期/>

### ✅ 胜诉模式（公司动了不该动的人）

| 公司操作 | 法院判定 |
|----------|----------|
| 医疗期内以"不胜任"或"裁员"解除 | 违反第 42 条 → 违法解除 2N |
| 三期女职工被调岗降薪 | 违反第 42 条 → 恢复原岗位 + 工资差额 |
| 哺乳期被辞退 | 双倍工资差额 + 违法解除赔偿 |

### ⚠️ 上海特殊规定

- 医疗期标准：根据**实际工龄 + 本单位工龄**确定，3-24 个月不等
- **延长有效期至 2030-06-30**（参考：<https://z.dasszss.com/references/laws/supplementary/11A-上海医疗期-延长2030/>）
- 三期女职工：产假 158 天（98 + 60 上海延长）

### ❌ 败诉模式

- 三期女职工**严重违纪**（39 条）→ 公司仍可解除
- 医疗期满仍不能从事原工作或另行安排的工作（40 条）→ 公司可以 N+1 解除

## 五、加班费

📖 完整 by-type 案例：<https://z.dasszss.com/references/cases/by-type/加班费/>

### ✅ 胜诉模式

| 关键证据 | 法院倾向 |
|----------|----------|
| **打卡记录（OA / 钉钉 / 企业微信）** | 双方核对，超时部分应付 |
| **工作群聊记录（晚上深夜布置工作 / 周末发任务）** | 视为延长工作时间证据 |
| **客户邮件 / 项目交付时间戳** | 推定加班事实 |
| **公司单方"实行综合工时制"但未报劳动行政部门审批** | 综合工时无效，按标准工时算加班 |

### ❌ 败诉模式

- **公司主张"不定时工作制"且经审批** → 一般不支持加班费（销售、高管常见）
- **没有任何加班记录证据**（钉钉记录被清空、邮件被收回）
- **明确签字"加班调休制度"且执行** → 调休可抵加班费

### 🎯 倍数

- 工作日加班：1.5 倍
- 休息日加班：2 倍（可调休抵）
- 法定假日加班：3 倍（**不可调休抵**）

## 六、欠薪 / 双倍工资差额（农民工/试用期常见）

### ✅ 胜诉率最高的两类

| 主张 | 法律依据 | 上限 |
|------|----------|------|
| **未签合同双倍工资** | 第 82 条 | 11 个月 |
| **拖欠工资加付赔偿金** | 第 85 条 | 25%-100% |

### ⏰ 时效规则（重要）

- 双倍工资：**从应当订立合同之日起 1 年**，**不是离职 1 年**
- 拖欠工资：**在职期间不受 1 年限制**，但离职后**1 年内**必须提起
