## [1.0.0] - 2025-04-10

### 🎉 正式发布

经过重构和增强，项目正式达到 1.0.0 版本，具备完整的劳动维权知识库体系。

### ✨ 新增功能

#### 文档体系重构
- **分层架构**：按维权阶段（预防→危机→证据→行动→开庭→后续）组织文档
- **新增模块**：
  - `docs/phase-0-prepare/job-offer-checklist.md` - 入职避雷指南
  - `docs/phase-2-evidence/recording-guide.md` - 录音取证指南
  - `resources/official-resources.md` - 官方资源导航

#### 实用工具
- **赔偿计算器** (`tools/compensation-calculator.py`)：
  - 自动计算经济补偿金（N）
  - 计算违法解除赔偿金（2N）
  - 上海三倍封顶自动判断
  - 加班费、年假补偿估算

#### 数据更新
- 更新为2024年度上海最新标准：
  - 月平均工资：12,434元/月
  - 三倍封顶：37,302元/月
  - 社保基数下限：7,460元/月

### 📁 文件组织

```
├── docs/              # 按维权阶段组织的实战指南
├── references/        # 法律法规、上海规定、赔偿计算
├── tools/            # 实用计算工具
└── resources/        # 官方资源导航
```

### 📊 内容统计

- 文档总数：14个专业指南
- 总字数：约30,000+字
- 覆盖法条：50+条核心法律条文
- 案例模板：10+份实战文书

### 🔧 技术改进

- 重新组织目录结构，提升可维护性
- 标准化文件命名和格式
- 增加快速开始指南

### ⚠️ 注意事项

- 所有内容基于上海地区2025-2026年标准
- 其他城市请参考当地规定
- 建议咨询专业律师获取针对性建议
