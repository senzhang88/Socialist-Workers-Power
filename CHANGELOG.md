## [1.1.0] - 2026-04-26

### 🎯 本次升级核心

把已离线整理的**判例知识库**、**法律全文**、**上海资源全栈**接入 Claude Skill，覆盖灯塔计划群体最高频维权场景，让 Skill（离线）与 [z.dasszss.com](https://z.dasszss.com/)（公网完整版）形成"先诊断、再深入"的闭环。

**Skill 体量从 1563 行 → 2796 行（+78%），场景模板从 3 个 → 9 个（+200%）**。

### ✨ 新增功能

#### 1. 判例知识库接入（核心）
- 新增 `skill/references/cases.md`（220 行）—— 按场景提炼**胜诉模式 / 败诉模式 / 关键证据清单**
  - 违法解除（胜率约 60%）
  - 软裁员·不提供劳动条件
  - 年终奖争议（含《民法典》第 159 条阻却条件成就武器）
  - 医疗期 / 三期女职工保护
  - 加班费
  - 欠薪 / 双倍工资差额
- 链接到公网完整版：50 篇胜诉深度分析 + 50 篇败诉深度分析 + 686 条上海判例索引（CSV/JSON）

#### 2. 6 大高频场景模板（灯塔计划主战场）
新增 `SKILL.md` 场景：
- **场景 4**：欠薪 / 拖欠工资（含监察 vs 仲裁两条战线选择）
- **场景 5**：年终奖争议（含历年发放推定 + 阻却条件成就）
- **场景 6**：加班费（含工时制识别 + 倍数计算 + 败诉雷区）
- **场景 7**：医疗期 / 工伤保护（含上海医疗期标准 3-24 个月分档）
- **场景 8**：三期女职工保护（孕期/产期/哺乳期 + 上海产假 158 天）
- **场景 9**：试用期辞退 / 未签合同双倍工资（含金额示例）

#### 3. 2025-2026 年法规时效更新
- ⚠️ **《劳动争议司法解释（二）》2025-09-01 生效**（与此前不一致的以解释二为准；解释一第 32 条第 1 款同时废止）
- ⚠️ **《上海市企业工资支付办法》现行有效至 2026-08-15**（人社局 2025-07-08 答复）
- ⚠️ **上海医疗期标准延长至 2030-06-30**（市府 2025-03-21 通知）

#### 4. 最高人民法院指导案例
- **指导案例 18 号**（中兴通讯诉王鹏）：违纪解除的**比例原则**
- **指导案例 183 号**（房玥诉大都会人寿）：**末位淘汰 / 模糊绩效辞退构成违法**（PIP 必备武器）

#### 5. 上海维权资源全栈
`skill/references/shanghai.md` 从 142 行 → 224 行：
- 上海市总工会（武康路 189 号）+ 12351 援助中心
- 法律援助申请条件 + 各区法律援助中心地址
- 街道司法所（最近的法援点）
- 信访通道（兜底）
- 一网通办 / 随申办 / 上海法网网上入口
- 上海市劳动保障监察总队 021-23116664 / 社保中心 021-962222

### 🚀 如何使用

#### 一键安装（推荐）

```bash
claude skills install https://github.com/senzhang88/Socialist-Workers-Power/releases/download/v1.1/work-powers-shanghai-v1.1.skill
```

#### 触发方式（直接对话）

```
公司让我停岗待业怎么办
帮我算一下违法解除的赔偿
试用期被辞了，公司没签合同
怀孕被辞退，怎么办
```

Skill 自动识别场景 → 检索对应的胜负判例模式 → 给出"今天/明天做什么 + 证据清单 + 联系方式"。

## [1.0.0] - 2025-03-15

### 🎉 正式发布

经过重构和增强，项目正式达到 1.0.0 版本，具备完整的劳动维权知识库体系。

### ✨ 新增功能

#### 文档体系重构
- **分层架构**：按维权阶段（预防→危机→证据→行动→开庭→后续）组织文档
- **新增模块**：
  - `docs/phase-0-prepare/job-offer-checklist.md` - 入职避雷指南
  - `docs/phase-2-evidence/recording-guide.md` - 录音取证指南
  - `resources/official-resources.md` - 官方资源导航

#### 实用工具
- **赔偿计算器** (`tools/compensation-calculator.py`)：
  - 自动计算经济补偿金（N）
  - 计算违法解除赔偿金（2N）
  - 上海三倍封顶自动判断
  - 加班费、年假补偿估算

#### 数据更新
- 更新为2024年度上海最新标准：
  - 月平均工资：12,434元/月
  - 三倍封顶：37,302元/月
  - 社保基数下限：7,460元/月

### 📁 文件组织

```
├── docs/              # 按维权阶段组织的实战指南
├── references/        # 法律法规、上海规定、赔偿计算
├── tools/            # 实用计算工具
└── resources/        # 官方资源导航
```
